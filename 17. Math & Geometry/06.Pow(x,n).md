# 50. Pow(x, n)
Medium

Implement pow(x, n), which calculates x raised to the power n (i.e., x^n).

 

Example 1:
```
Input: x = 2.00000, n = 10
Output: 1024.00000
```

Constraints:
- -2^31 <= n <= 2^31-1

## 解題思路

快速冪（Binary Exponentiation / Divide and Conquer）：

1. **基礎情況**：
   - 若 `x == 0`，回傳 0。
   - 若 `n == 0`，回傳 1。
2. **處理負指數**：若 `n < 0`，則 `x^n = (1/x)^(-n)`。
3. **分治遞迴**：
   - 計算 `res = pow(x, n // 2)`。
   - 若 `n` 為偶數，結果為 `res * res`。
   - 若 `n` 為奇數，結果為 `x * res * res`。

時間複雜度：O(log N)，空間複雜度：O(log N)（遞迴深度）。

```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        def helper(x, n):
            if x == 0:
                return 0
            if n == 0:
                return 1

            res = helper(x, n // 2)
            res = res * res
            return x * res if n % 2 else res

        res = helper(x, abs(n))
        return res if n >= 0 else 1 / res
```

## Complexity Analysis
- Time Complexity: O(log n)
- Space Complexity: O(log n)
