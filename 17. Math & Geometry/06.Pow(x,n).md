# [50. Pow(x, n)](https://leetcode.com/problems/powx-n/)
Medium

Implement pow(x, n), which calculates x raised to the power n (i.e., x^n).

 

Example 1:
```
Input: x = 2.00000, n = 10
Output: 1024.00000
```
Example 2:
```
Input: x = 2.10000, n = 3
Output: 9.26100
```
Example 3:
```
Input: x = 2.00000, n = -2
Output: 0.25000
 ```

Constraints:

- -100.0 < x < 100.0
- -2^31 <= n <= 2^31-1

## 解題思路

快速冪（Binary Exponentiation）：

- 每次將 n 除以 2，若為奇數則額外乘一次 x
- n 為負時，取倒數 `x = 1/x`，n 取絕對值
- 注意 n = INT_MIN 時取絕對值會溢位，用 `long` 處理

時間複雜度：O(log N)，空間複雜度：O(1)。

```cpp
class Solution {
public:
    double myPow(double x, int n) {
        long exp = n;
        if (exp < 0) { x = 1.0 / x; exp = -exp; }
        double result = 1.0;
        while (exp > 0) {
            if (exp % 2 == 1) result *= x;
            x *= x;
            exp /= 2;
        }
        return result;
    }
};
```

## Complexity Analysis
- Time Complexity: O(log N)
- Space Complexity: O(1)
