# 48. Rotate Image
Medium

You are given an n x n 2D matrix representing an image, rotate the image by 90 degrees (clockwise). You have to rotate the image in-place.

 

Example 1:
```
Input: matrix = [[1,2,3],[4,5,6],[7,8,9]]
Output: [[7,4,1],[8,5,2],[9,6,3]]
```

Constraints:
- n == matrix.length == matrix[i].length

## 解題思路

順時針旋轉 90° 的步驟：
1. **上下翻轉 (Reverse)**：將矩陣的第一列與最後一列交換，第二列與倒數第二列交換...
2. **轉置 (Transpose)**：將 `matrix[i][j]` 與 `matrix[j][i]` 交換。

或者更直觀的單一循環法：
- 旋轉每一層（矩形環）。
- 對於每一層，將四個角對應的元素進行四向交換。

時間複雜度：O(N²)，空間複雜度：O(1)。

```python
class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        l, r = 0, len(matrix) - 1
        while l < r:
            for i in range(r - l):
                top, bottom = l, r

                # save the topleft
                topLeft = matrix[top][l + i]

                # move bottom left into top left
                matrix[top][l + i] = matrix[bottom - i][l]

                # move bottom right into bottom left
                matrix[bottom - i][l] = matrix[bottom][r - i]

                # move top right into bottom right
                matrix[bottom][r - i] = matrix[top + i][r]

                # move top left into top right
                matrix[top + i][r] = topLeft
            r -= 1
            l += 1
```

## Complexity Analysis
- Time Complexity: O(n^2)
- Space Complexity: O(1)
