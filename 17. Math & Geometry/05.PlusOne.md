# [66. Plus One](https://leetcode.com/problems/plus-one/)
Easy

You are given a large integer represented as an integer array digits, where each digits[i] is the ith digit of the integer. Increment the large integer by one and return the resulting array of digits.

 

Example 1:
```
Input: digits = [1,2,3]
Output: [1,2,4]
```
Example 2:
```
Input: digits = [4,3,2,1]
Output: [4,3,2,2]
```
Example 3:
```
Input: digits = [9]
Output: [1,0]
 ```

Constraints:

- 1 <= digits.length <= 100
- 0 <= digits[i] <= 9
- digits does not contain any leading 0's.

## 解題思路

從最低位開始加 1：

- 若當前位 < 9，直接 +1 回傳
- 否則將當前位設為 0，繼續進位
- 全部都是 9 的情況：迴圈結束後在最前面插入 1

時間複雜度：O(N)，空間複雜度：O(1)（最壞情況 O(N) 建新陣列）。

```cpp
class Solution {
public:
    vector<int> plusOne(vector<int>& digits) {
        int n = digits.size();
        for (int i = n - 1; i >= 0; i--) {
            if (digits[i] < 9) {
                digits[i]++;
                return digits;
            }
            digits[i] = 0;
        }
        digits.insert(digits.begin(), 1);
        return digits;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N)
- Space Complexity: O(1)
