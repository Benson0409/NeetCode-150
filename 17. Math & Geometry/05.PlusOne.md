# 66. Plus One
Easy

You are given a large integer represented as an integer array digits. Increment the large integer by one and return the resulting array of digits.

 

Example 1:
```
Input: digits = [1,2,3]
Output: [1,2,4]
```

Constraints:
- 1 <= digits.length <= 100

## 解題思路

從最低位（陣列末尾）開始處理：

1. 遍歷陣列從右往左：
   - 若當前數字小於 9，則直接加 1 並回傳結果。
   - 若當前數字等於 9，將其變為 0，並繼續向左進位。
2. 特殊情況：若陣列所有位數都是 9（如 [9, 9, 9]）：
   - 迴圈結束後陣列會變成 [0, 0, 0]。
   - 在陣列開頭插入 1，變成 [1, 0, 0, 0]。

時間複雜度：O(N)，空間複雜度：O(1)（若不計入結果陣列的擴充）。

```python
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        for i in range(len(digits) - 1, -1, -1):
            if digits[i] < 9:
                digits[i] += 1
                return digits
            digits[i] = 0

        digits.insert(0, 1)
        return digits
```

## Complexity Analysis
- Time Complexity: O(n)
- Space Complexity: O(n)
