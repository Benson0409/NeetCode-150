# 73. Set Matrix Zeroes
Medium

Given an m x n integer matrix matrix, if an element is 0, set its entire row and column to 0's. You must do it in place.

 

Example 1:
```
Input: matrix = [[1,1,1],[1,0,1],[1,1,1]]
Output: [[1,0,1],[0,0,0],[1,0,1]]
```

Constraints:
- m == matrix.length
- n == matrix[0].length

## 解題思路

O(1) 空間解法：利用第一行與第一列來記錄該行/該列是否應清零。

1. **標記階段**：
   - 使用一個變數 `rowZero` 記錄第一行是否本來就有 0。
   - 遍歷整個矩陣，若 `matrix[r][c] == 0`：
     - 將該行首 `matrix[r][0]` 設為 0。
     - 將該列首 `matrix[0][c]` 設為 0（若 `r > 0`，否則標記 `rowZero = True`）。
2. **填充階段**（除了第一行第一列）：
   - 遍歷從 `(1, 1)` 到 `(m-1, n-1)`，若其對應的行首或列首為 0，則將當前元素設為 0。
3. **處理第一列**：
   - 若 `matrix[0][0] == 0`，將第一列全部設為 0。
4. **處理第一行**：
   - 若 `rowZero == True`，將第一行全部設為 0。

時間複雜度：O(m × n)，空間複雜度：O(1)。

```python
class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        ROWS, COLS = len(matrix), len(matrix[0])
        rowZero = False

        # determine which rows/cols need to be zero
        for r in range(ROWS):
            for c in range(COLS):
                if matrix[r][c] == 0:
                    matrix[0][c] = 0
                    if r > 0:
                        matrix[r][0] = 0
                    else:
                        rowZero = True

        for r in range(1, ROWS):
            for c in range(1, COLS):
                if matrix[0][c] == 0 or matrix[r][0] == 0:
                    matrix[r][c] = 0

        if matrix[0][0] == 0:
            for r in range(ROWS):
                matrix[r][0] = 0

        if rowZero:
            for c in range(COLS):
                matrix[0][c] = 0
```

## Complexity Analysis
- Time Complexity: O(m * n)
- Space Complexity: O(1)
