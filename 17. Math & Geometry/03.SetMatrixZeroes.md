# [73. Set Matrix Zeroes](https://leetcode.com/problems/set-matrix-zeroes/)
Medium

Given an m x n integer matrix matrix, if an element is 0, set its entire row and column to 0's. You must do it in place.

 

Example 1:
```
Input: matrix = [[1,1,1],[1,0,1],[1,1,1]]
Output: [[1,0,1],[0,0,0],[1,0,1]]
```
Example 2:
```
Input: matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
Output: [[0,0,0,0],[0,4,5,0],[0,3,1,0]]
 ```

Constraints:

- m == matrix.length
- n == matrix[0].length
- 1 <= m, n <= 200

## 解題思路

O(1) 空間解法：利用第一行與第一列做為標記陣列：

1. 先記錄第一列是否含 0（用 `col0` 變數）
2. 掃描 `matrix[0][j]`，若有 0 標記 `matrix[0][0] = 0`（代表第一行需要清零）
3. 從 `(1,1)` 開始掃描，若 `matrix[i][j] == 0`，標記 `matrix[i][0] = 0` 和 `matrix[0][j] = 0`
4. 再次從 `(1,1)` 掃描，依據標記清零
5. 最後處理第一行（依 `matrix[0][0]`）和第一列（依 `col0`）

時間複雜度：O(m × n)，空間複雜度：O(1)。

```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int m = matrix.size(), n = matrix[0].size();
        int col0 = 1;

        // Check first column
        for (int i = 0; i < m; i++)
            if (matrix[i][0] == 0) col0 = 0;

        // Check first row -> mark matrix[0][0]
        for (int j = 0; j < n; j++)
            if (matrix[0][j] == 0) matrix[0][0] = 0;

        // Mark rows/cols using first row/col
        for (int i = 1; i < m; i++)
            for (int j = 1; j < n; j++)
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }

        // Zero out based on marks
        for (int i = 1; i < m; i++)
            for (int j = 1; j < n; j++)
                if (matrix[i][0] == 0 || matrix[0][j] == 0)
                    matrix[i][j] = 0;

        // Handle first row
        if (matrix[0][0] == 0)
            for (int j = 1; j < n; j++) matrix[0][j] = 0;

        // Handle first column
        if (col0 == 0)
            for (int i = 0; i < m; i++) matrix[i][0] = 0;
    }
};
```

## Complexity Analysis
- Time Complexity: O(m × n)
- Space Complexity: O(1)
