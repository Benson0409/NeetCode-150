# 202. Happy Number
Easy

Write an algorithm to determine if a number n is happy. Return true if n is a happy number, and false if not.

 

Example 1:
```
Input: n = 19
Output: true
```

Constraints:
- 1 <= n <= 2^31 - 1

## 解題思路

方法：使用集合 (Set) 偵測循環。

1. 按照快樂數字的定義計算每位數字的平方和。
2. 使用一個集合 `visit` 記錄所有出現過的數字。
3. 重複計算：
   - 若計算結果為 1，則是快樂數字。
   - 若結果出現在 `visit` 中，代表進入了無限循環且不包含 1，回傳 `False`。
   - 將結果加入 `visit` 繼續下一輪。

也可以使用 Floyd 快慢指標來偵測循環，以達成 O(1) 空間複雜度。

時間複雜度：O(log n)，空間複雜度：O(log n)。

```python
class Solution:
    def isHappy(self, n: int) -> bool:
        visit = set()

        while n not in visit:
            visit.add(n)
            n = self.sumOfSquares(n)
            if n == 1:
                return True
        return False

    def sumOfSquares(self, n: int) -> int:
        output = 0
        while n:
            digit = n % 10
            digit = digit**2
            output += digit
            n = n // 10
        return output
```

## Complexity Analysis
- Time Complexity: O(log n)
- Space Complexity: O(log n)
