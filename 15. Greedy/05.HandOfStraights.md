# [846. Hand of Straights](https://leetcode.com/problems/hand-of-straights/)
Medium

Alice has some number of cards and she wants to rearrange the cards into groups so that each group is of size groupSize, and consists of groupSize consecutive cards.

Given an integer array hand where hand[i] is the value written on the ith card and an integer groupSize, return true if she can rearrange the cards, or false otherwise.

 

Example 1:
```
Input: hand = [1,2,3,6,2,3,4,7,8], groupSize = 3
Output: true
Explanation: Alice's hand can be rearranged as [1,2,3],[2,3,4],[6,7,8]
```
Example 2:
```
Input: hand = [1,2,3,4,5], groupSize = 4
Output: false
``` 

Constraints:

- 1 <= hand.length <= 10^4
- 0 <= hand[i] <= 10^9
- 1 <= groupSize <= hand.length

## 解題思路

貪心 + 有序 map：

1. 若牌總數不能被 groupSize 整除，直接 false
2. 用 `map<int,int>` 統計每張牌的數量（有序，可按升序遍歷）
3. 每次從最小的牌開始，嘗試組成連續 groupSize 張的群組
4. 若某張中間的牌不存在或數量不足，回傳 false

時間複雜度：O(N log N)，空間複雜度：O(N)。

```cpp
class Solution {
public:
    bool isNStraightHand(vector<int>& hand, int groupSize) {
        if (hand.size() % groupSize != 0) return false;

        map<int, int> cnt;
        for (int c : hand) cnt[c]++;

        for (auto& [val, freq] : cnt) {
            if (freq == 0) continue;
            int times = freq;
            for (int i = 0; i < groupSize; i++) {
                if (cnt[val + i] < times) return false;
                cnt[val + i] -= times;
            }
        }
        return true;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N log N)
- Space Complexity: O(N)
