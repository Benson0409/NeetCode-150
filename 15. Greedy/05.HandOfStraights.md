# 846. Hand of Straights
Medium

Alice has some number of cards and she wants to rearrange the cards into groups so that each group is of size groupSize, and consists of groupSize consecutive cards. Return true if she can, false otherwise.

 

Example 1:
```
Input: hand = [1,2,3,6,2,3,4,7,8], groupSize = 3
Output: true
```

Constraints:
- 1 <= hand.length <= 10^4

## 解題思路

貪心 + 雜湊表：

1. 若牌數不能被 `groupSize` 整除，直接回傳 `False`。
2. 使用雜湊表（`Counter`）統計每張牌出現的次數。
3. 使用最小堆（Min-Heap）來存儲所有獨特的牌，以便每次都能從最小的牌開始處理。
4. 每次從堆中彈出最小的牌 `first`：
   - 若該牌次數為 0，跳過。
   - 否則，嘗試組成一組連續的牌：`first, first + 1, ..., first + groupSize - 1`。
   - 在雜湊表中減少這些牌的計數。若某張中間的牌不存在或次數已為 0，則無法組成，回傳 `False`。
5. 重複直到所有牌都處理完。

時間複雜度：O(N log N)，空間複雜度：O(N)。

```python
import heapq
from collections import Counter

class Solution:
    def isNStraightHand(self, hand: List[int], groupSize: int) -> bool:
        if len(hand) % groupSize != 0:
            return False

        count = Counter(hand)
        minH = list(count.keys())
        heapq.heapify(minH)

        while minH:
            first = minH[0]
            if count[first] == 0:
                heapq.heappop(minH)
                continue

            for i in range(first, first + groupSize):
                if count[i] == 0:
                    return False
                count[i] -= 1
        return True
```

## Complexity Analysis
- Time Complexity: O(n * log(n))
- Space Complexity: O(n)
