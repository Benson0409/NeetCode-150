# 53. Maximum Subarray
Easy

Given an integer array nums, find the contiguous subarray (containing at least one number) which has the largest sum and return its sum.

 

Example 1:
```
Input: nums = [-2,1,-3,4,-1,2,1,-5,4]
Output: 6
Explanation: [4,-1,2,1] has the largest sum = 6.
```

Constraints:
- 1 <= nums.length <= 10^5

## 解題思路

Kadane's Algorithm（貪心法）：

1. 維護一個「當前子陣列的加總」`curSum`。
2. 遍歷陣列中的每個數字 `n`：
   - 若 `curSum` 已小於 0，則捨棄它，從 `n` 重新開始累積（因為負的前綴只會讓後面的總和變小）。
   - 將 `n` 加入 `curSum`。
   - 同時更新全域的最大加總 `maxSub`。

時間複雜度：O(N)，空間複雜度：O(1)。

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        maxSub = nums[0]
        curSum = 0

        for n in nums:
            if curSum < 0:
                curSum = 0
            curSum += n
            maxSub = max(maxSub, curSum)
        return maxSub
```

## Complexity Analysis
- Time Complexity: O(n)
- Space Complexity: O(1)
