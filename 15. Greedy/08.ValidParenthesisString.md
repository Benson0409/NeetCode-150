# 678. Valid Parenthesis String
Medium

Given a string s containing only '(', ')' and '*', return true if s is valid. Rules:
- '(' must have a matching ')'.
- '*' can be '(', ')', or "".

 

Example 1:
```
Input: s = "(*)"
Output: true
```

Constraints:
- 1 <= s.length <= 100

## 解題思路

貪心法（追蹤未匹配左括號的可能範圍）：

1. 我們用 `leftMin` 和 `leftMax` 記錄當前位置「可能剩餘」的左括號數量範圍。
2. 遍歷字串：
   - 若為 `(`：`leftMin += 1`, `leftMax += 1`。
   - 若為 `)`：`leftMin -= 1`, `leftMax -= 1`。
   - 若為 `*`：`leftMin -= 1` (當作 `)`), `leftMax += 1` (當作 `(`)。
3. 在過程中：
   - 若 `leftMax < 0`：代表即使把所有 `*` 都當作 `(`，右括號還是太多，回傳 `False`。
   - 若 `leftMin < 0`：將其重置為 0（因為我們不能有負數個左括號，此時代表我們把某些 `*` 當成了空字串）。
4. 最終檢查 `leftMin == 0` 是否成立。

時間複雜度：O(N)，空間複雜度：O(1)。

```python
class Solution:
    def checkValidString(self, s: str) -> bool:
        leftMin, leftMax = 0, 0

        for c in s:
            if c == "(":
                leftMin, leftMax = leftMin + 1, leftMax + 1
            elif c == ")":
                leftMin, leftMax = leftMin - 1, leftMax - 1
            else:
                leftMin, leftMax = leftMin - 1, leftMax + 1

            if leftMax < 0:
                return False
            if leftMin < 0:  # s = ( * )
                leftMin = 0
        return leftMin == 0
```

## Complexity Analysis
- Time Complexity: O(n)
- Space Complexity: O(1)
