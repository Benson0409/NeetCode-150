# [678. Valid Parenthesis String](https://leetcode.com/problems/valid-parenthesis-string/)
Medium

Given a string s containing only three types of characters: '(', ')' and '*', return true if s is valid.

The following rules define a valid string:

- Any left parenthesis '(' must have a corresponding right parenthesis ')'.
- Any right parenthesis ')' must have a corresponding left parenthesis '('.
- Left parenthesis '(' must go before the corresponding right parenthesis ')'.
- '*' could be treated as '(', ')' or empty string "".
 

Example 1:
```
Input: s = "()"
Output: true
```
Example 2:
```
Input: s = "(*)"
Output: true
```
Example 3:
```
Input: s = "(*))"
Output: true
 ```

Constraints:

- 1 <= s.length <= 100
- s[i] is '(', ')' or '*'.

## 解題思路

貪心法（追蹤可能的未匹配左括號數量範圍）：

- `leftMin`：最少未匹配 `(` 數（`*` 優先當 `)` 或空字串）
- `leftMax`：最多未匹配 `(` 數（`*` 優先當 `(`）
- 遇到 `(`：兩者 +1；遇到 `)`：兩者 -1；遇到 `*`：min-1, max+1
- 若 `leftMax < 0`，代表 `)` 太多，必然無效
- `leftMin` 不能為負（`*` 不能當多餘的 `)`），夾在 0 以上
- 最終 `leftMin == 0` 代表有一種合法的解釋

時間複雜度：O(N)，空間複雜度：O(1)。

```cpp
class Solution {
public:
    bool checkValidString(string s) {
        int leftMin = 0, leftMax = 0;
        for (char c : s) {
            if (c == '(') { leftMin++; leftMax++; }
            else if (c == ')') { leftMin--; leftMax--; }
            else { leftMin--; leftMax++; }  // '*'
            if (leftMax < 0) return false;
            if (leftMin < 0) leftMin = 0;
        }
        return leftMin == 0;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N)
- Space Complexity: O(1)
