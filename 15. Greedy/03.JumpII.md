# 45. Jump Game II
Medium

Given an array of non-negative integers nums, you are initially positioned at the first index. Each element represents your maximum jump length at that position. Your goal is to reach the last index in the minimum number of jumps. You can assume that you can always reach the last index.

 

Example 1:
```
Input: nums = [2,3,1,1,4]
Output: 2
```

Constraints:
- 1 <= nums.length <= 10^4

## 解題思路

貪心法（區間思維）：

1. 我們將跳躍過程看作是一層一層的區間。
2. `l` 和 `r` 代表當前跳躍次數能到達的範圍（左界與右界）。
3. 對於當前區間 `[l, r]`，遍歷其中的每個位置，計算下一步能跳到的最遠位置 `farthest`。
4. 更新區間：下一層的 `l = r + 1`，`r = farthest`。
5. 每更新一次區間，跳躍次數 `res` 加 1。
6. 當 `r` 到達或超過最後一個索引時停止。

時間複雜度：O(N)，空間複雜度：O(1)。

```python
class Solution:
    def jump(self, nums: List[int]) -> int:
        res = 0
        l = r = 0

        while r < len(nums) - 1:
            farthest = 0
            for i in range(l, r + 1):
                farthest = max(farthest, i + nums[i])
            l = r + 1
            r = farthest
            res += 1
        return res
```

## Complexity Analysis
- Time Complexity: O(n)
- Space Complexity: O(1)
