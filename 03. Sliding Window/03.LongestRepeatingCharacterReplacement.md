# 424. Longest Repeating Character Replacement
Medium

You are given a string s and an integer k. You can choose any character of the string and change it to any other uppercase English character. You can perform this operation at most k times.

Return the length of the longest substring containing the same letter you can get after performing the above operations.

 

Example 1:
```
Input: s = "ABAB", k = 2
Output: 4
Explanation: Replace the two 'A's with two 'B's or vice versa.
```
Example 2:
```
Input: s = "AABABBA", k = 1
Output: 4
Explanation: Replace the one 'A' in the middle with 'B' and form "AABBBBA".
The substring "BBBB" has the longest repeating letters, which is 4.
``` 

Constraints:

- 1 <= s.length <= 10<sup>5</sup>
- s consists of only uppercase English letters.
- 0 <= k <= s.length

## 解題思路

滑動窗口核心概念：

- 窗口內「需要替換的字元數」= 窗口長度 - 窗口內出現最多次的字元數（`mostRepeatingCharCount`）
- 若需替換數 > k，則縮小窗口左端（`windowStart++`），同時更新字元頻率 map
- `mostRepeatingCharCount` 只需累計（不需回退），因為我們只在意最大窗口是否合法

時間複雜度：O(n)，空間複雜度：O(26) = O(1)。

```cpp
class Solution {
public:
    int characterReplacement(string s, int k) {
        unordered_map<char, int> charFreq;
        int windowStart = 0, maxLength = 0, mostRepeating = 0;

        for (int windowEnd = 0; windowEnd < (int)s.size(); windowEnd++) {
            char right = s[windowEnd];
            charFreq[right]++;
            mostRepeating = max(mostRepeating, charFreq[right]);

            while (windowEnd - windowStart + 1 - mostRepeating > k) {
                char left = s[windowStart];
                charFreq[left]--;
                if (charFreq[left] == 0)
                    charFreq.erase(left);
                windowStart++;
            }
            maxLength = max(maxLength, windowEnd - windowStart + 1);
        }

        return maxLength;
    }
};
```
