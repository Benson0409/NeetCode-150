# 3. Longest Substring Without Repeating Characters
Medium

Given a string s, find the length of the longest substring without repeating characters.

 

Example 1:
```
Input: s = "abcabcbb"
Output: 3
Explanation: The answer is "abc", with the length of 3.
```
Example 2:
```
Input: s = "bbbbb"
Output: 1
Explanation: The answer is "b", with the length of 1.
```
Example 3:
```
Input: s = "pwwkew"
Output: 3
Explanation: The answer is "wke", with the length of 3.
Notice that the answer must be a substring, "pwke" is a subsequence and not a substring.
 ```

Constraints:

- 0 <= s.length <= 5 * 104
- s consists of English letters, digits, symbols and spaces.

## 解題思路

使用滑動窗口（可變大小）搭配 HashMap 記錄每個字元最後出現的位置：

- `winStart` 為窗口左端
- 遍歷 `winEnd`，若當前字元已在 map 中，將 `winStart` 更新為 `max(winStart, 上次出現位置 + 1)`（確保不往左退）
- 更新 map 中字元的最新位置
- 每步更新最大長度

時間複雜度：O(n)，空間複雜度：O(min(n, 字元集大小))。

```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_map<char, int> mp;
        int maxLen = 0, winStart = 0;

        for (int winEnd = 0; winEnd < (int)s.size(); winEnd++) {
            char ch = s[winEnd];
            if (mp.count(ch))
                winStart = max(winStart, mp[ch] + 1);
            mp[ch] = winEnd;
            maxLen = max(maxLen, winEnd - winStart + 1);
        }
        return maxLen;
    }
};
```
