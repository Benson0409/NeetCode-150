# [239. Sliding Window Maximum](https://leetcode.com/problems/sliding-window-maximum/)
Hard

You are given an array of integers nums, there is a sliding window of size k which is moving from the very left of the array to the very right. You can only see the k numbers in the window. Each time the sliding window moves right by one position.

Return the max sliding window.

 

Example 1:
```
Input: nums = [1,3,-1,-3,5,3,6,7], k = 3
Output: [3,3,5,5,6,7]
Explanation: 
Window position                Max
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7
```
Example 2:
```
Input: nums = [1], k = 1
Output: [1]
``` 

Constraints:
- 1 <= nums.length <= 10<sup>5</sup>
- -10<sup>4</sup> <= nums[i] <= 10<sup>4</sup>
- 1 <= k <= nums.length

## 解題思路

使用單調遞減的雙端佇列（Deque）存放「索引」：

1. **移除過期元素**：若隊頭索引已超出窗口範圍（`<= wE - k`），從隊頭移除。
2. **維持單調遞減**：若隊尾對應的值 ≤ 當前元素，從隊尾移除（舊的小值對未來窗口沒用）。
3. 將當前索引加入隊尾。
4. 當窗口形成（`wE >= k-1`），隊頭就是當前窗口的最大值索引。

時間複雜度：O(n)（每個元素最多入隊/出隊各一次），空間複雜度：O(k)。

```cpp
class Solution {
public:
    vector<int> maxSlidingWindow(vector<int>& nums, int k) {
        vector<int> res;
        deque<int> q; // 存索引，單調遞減（隊頭最大）
        int wS = 0;

        for (int wE = 0; wE < (int)nums.size(); wE++) {
            // 移除超出窗口的隊頭
            while (!q.empty() && q.front() <= wE - k)
                q.pop_front();

            // 維持單調遞減：移除比當前元素小的隊尾
            while (!q.empty() && nums[q.back()] <= nums[wE])
                q.pop_back();

            q.push_back(wE);

            // 窗口形成後輸出最大值
            if (wE >= k - 1)
                res.push_back(nums[q.front()]);
        }
        return res;
    }
};
```

## Complexity Analysis
- Time Complexity: O(n)
- Space Complexity: O(k), where k < n
