# 567. Permutation in String
Medium

Given two strings s1 and s2, return true if s2 contains a permutation of s1, or false otherwise.

In other words, return true if one of s1's permutations is the substring of s2.

 

Example 1:
```
Input: s1 = "ab", s2 = "eidbaooo"
Output: true
Explanation: s2 contains one permutation of s1 ("ba").
```
Example 2:
```
Input: s1 = "ab", s2 = "eidboaoo"
Output: false
 ```

Constraints:

- 1 <= s1.length, s2.length <= 10<sup>4</sup>
- s1 and s2 consist of lowercase English letters.

## 解題思路

使用固定大小為 `s1.length()` 的滑動窗口，搭配兩個大小為 26 的計數陣列：

- `s1Map`：s1 的字母頻率
- `s2Map`：當前窗口的字母頻率
- `matches`：兩陣列中計數相同的字母數（共 26 個字母）

初始化第一個窗口後，每次滑動時：
1. 加入右端新字元 → 更新 `s2Map` 與 `matches`
2. 移除左端舊字元 → 更新 `s2Map` 與 `matches`
3. 若 `matches == 26`，代表找到排列

時間複雜度：O(s2.length())，空間複雜度：O(26) = O(1)。

```cpp
class Solution {
public:
    bool checkInclusion(string s1, string s2) {
        if (s1.size() > s2.size()) return false;

        int s1Map[26] = {}, s2Map[26] = {};
        for (int i = 0; i < (int)s1.size(); i++) {
            s1Map[s1[i] - 'a']++;
            s2Map[s2[i] - 'a']++;
        }

        int matches = 0;
        for (int i = 0; i < 26; i++)
            if (s1Map[i] == s2Map[i]) matches++;

        int windowStart = 0;
        for (int windowEnd = (int)s1.size(); windowEnd < (int)s2.size(); windowEnd++) {
            if (matches == 26) return true;

            int r = s2[windowEnd] - 'a';
            s2Map[r]++;
            if (s1Map[r] == s2Map[r])       matches++;
            else if (s1Map[r] + 1 == s2Map[r]) matches--;

            int l = s2[windowStart] - 'a';
            s2Map[l]--;
            if (s1Map[l] == s2Map[l])       matches++;
            else if (s1Map[l] - 1 == s2Map[l]) matches--;

            windowStart++;
        }

        return matches == 26;
    }
};
```

## Complexity Analysis
- Time Complexity: O(s2.length())
- Space Complexity: O(26) ~ O(1)
