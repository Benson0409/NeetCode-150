# [76. Minimum Window Substring](https://leetcode.com/problems/minimum-window-substring/)
Hard

Given two strings s and t of lengths m and n respectively, return the minimum window substring of s such that every character in t (including duplicates) is included in the window. If there is no such substring, return the empty string "".

The testcases will be generated such that the answer is unique.

A substring is a contiguous sequence of characters within the string.

 

Example 1:
```
Input: s = "ADOBECODEBANC", t = "ABC"
Output: "BANC"
Explanation: The minimum window substring "BANC" includes 'A', 'B', and 'C' from string t.
```
Example 2:
```
Input: s = "a", t = "a"
Output: "a"
Explanation: The entire string s is the minimum window.
```
Example 3:
```
Input: s = "a", t = "aa"
Output: ""
Explanation: Both 'a's from t must be included in the window.
Since the largest window of s only has one 'a', return empty string.
``` 

Constraints:
- m == s.length
- n == t.length
- 1 <= m, n <= 105
- s and t consist of uppercase and lowercase English letters.

## 解題思路

**方法一（基本）**：擴大窗口直到包含 t 的所有字元，再縮小窗口到最小，不斷更新最小長度。每次縮小時需呼叫 satisfy 函式，複雜度 O(N×K)。

**方法二（最佳化）**：用 `matched` 計數器取代每次呼叫 satisfy：
- `charFreqMap` 記錄 t 中字元的需求量
- 每當 map 中某字元需求降到 0（完全滿足），`matched++`
- 當 `matched == charFreqMap.size()`，表示窗口涵蓋所有需求 → 更新結果 → 縮小窗口

時間複雜度：O(N)，空間複雜度：O(K)，K 為 t 的字元種類數。

```cpp
// 方法一
class Solution {
    bool satisfy(unordered_map<char,int>& wMp, unordered_map<char,int>& tMp) {
        for (auto& [c, cnt] : tMp)
            if (!wMp.count(c) || wMp[c] < cnt) return false;
        return true;
    }
public:
    string minWindow(string s, string t) {
        int winS = 0, winE = 0;
        string ans;
        unordered_map<char,int> tMp, wMp;
        for (char c : t) tMp[c]++;

        while (winS < (int)s.size() && winE < (int)s.size()) {
            wMp[s[winE]]++;
            while (winS <= winE && satisfy(wMp, tMp)) {
                string cur = s.substr(winS, winE - winS + 1);
                if (ans.empty() || cur.size() < ans.size()) ans = cur;
                if (--wMp[s[winS]] == 0) wMp.erase(s[winS]);
                winS++;
            }
            winE++;
        }
        return ans;
    }
};
```

```cpp
// 方法二（最佳化）
class Solution {
public:
    string minWindow(string s, string t) {
        unordered_map<char,int> charFreq;
        for (char c : t) charFreq[c]++;

        int windowStart = 0, matched = 0, minLen = INT_MAX;
        string res;

        for (int windowEnd = 0; windowEnd < (int)s.size(); windowEnd++) {
            char right = s[windowEnd];
            if (charFreq.count(right)) {
                if (--charFreq[right] == 0) matched++;
            }

            while (matched == (int)charFreq.size()) {
                if (windowEnd - windowStart + 1 < minLen) {
                    minLen = windowEnd - windowStart + 1;
                    res = s.substr(windowStart, minLen);
                }
                char left = s[windowStart];
                if (charFreq.count(left)) {
                    if (charFreq[left]++ == 0) matched--;
                }
                windowStart++;
            }
        }
        return res;
    }
};
```

## Complexity Analysis
- 方法一：Time O(N×K)，Space O(K)
- 方法二：Time O(N)，Space O(K)
