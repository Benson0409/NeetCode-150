# 22. Generate Parentheses
Medium

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

> Feels like mismatch for the pattern stack

Example 1:
```
Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
```
Example 2:
```
Input: n = 1
Output: ["()"]
 ```

Constraints:

- 1 <= n <= 8

## 解題思路

使用回溯法（Backtracking）：

- 任何時候都可以加入 `(`，只要左括號數量 < n
- 只有當左括號數量 > 右括號數量時，才能加入 `)`
- 當左右括號數量都等於 n，加入結果

這樣確保每個生成的字串都是合法的括號組合。

時間複雜度：O(4^n / √n)（Catalan 數），空間複雜度：O(n)（遞迴深度）。

```cpp
class Solution {
    void generate(int n, int lCount, int rCount, string& op, vector<string>& ans) {
        if (lCount == n && rCount == n) {
            ans.push_back(op);
            return;
        }
        if (lCount < n) {
            op.push_back('(');
            generate(n, lCount + 1, rCount, op, ans);
            op.pop_back();
        }
        if (rCount < lCount) {
            op.push_back(')');
            generate(n, lCount, rCount + 1, op, ans);
            op.pop_back();
        }
    }

public:
    vector<string> generateParenthesis(int n) {
        vector<string> ans;
        string op;
        generate(n, 0, 0, op, ans);
        return ans;
    }
};
```
