# 20. Valid Parentheses
Easy

Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid.

An input string is valid if:

- Open brackets must be closed by the same type of brackets.
- Open brackets must be closed in the correct order.
 

Example 1:
```
Input: s = "()"
Output: true
```
Example 2:
```
Input: s = "()[]{}"
Output: true
```
Example 3:
```
Input: s = "(]"
Output: false
 ```

Constraints:

- 1 <= s.length <= 104
- s consists of parentheses only '()[]{}'.

## 解題思路

使用 Stack：

- 建立一個字典將右括號映射到對應的左括號。
- 遇到左括號 → 壓入 stack。
- 遇到右括號 → 若 stack 為空或頂端不是對應的左括號 → 回傳 `False`，否則彈出頂端。
- 遍歷結束後 stack 必須為空（否則有未關閉的左括號）。

時間複雜度：O(n)，空間複雜度：O(n)。

```python
class Solution:
    def isValid(self, s: str) -> bool:
        Map = {")": "(", "]": "[", "}": "{"}
        stack = []

        for c in s:
            if c not in Map:
                stack.append(c)
                continue
            if not stack or stack[-1] != Map[c]:
                return False
            stack.pop()

        return not stack
```
