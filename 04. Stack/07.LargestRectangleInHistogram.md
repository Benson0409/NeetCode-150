# [84. Largest Rectangle in Histogram](https://leetcode.com/problems/largest-rectangle-in-histogram/)
Hard

Given an array of integers heights representing the histogram's bar height where the width of each bar is 1, return the area of the largest rectangle in the histogram.

 

Example 1:
```
Input: heights = [2,1,5,6,2,3]
Output: 10
Explanation: The above is a histogram where width of each bar is 1.
The largest rectangle is shown in the red area, which has an area = 10 units.
```
Example 2:
```
Input: heights = [2,4]
Output: 4
 ```

Constraints:

- 1 <= heights.length <= 105
- 0 <= heights[i] <= 104

## 解題思路

使用單調遞增 Stack 存放 `{起始索引, 高度}`：

- 遍歷每根柱子，若當前高度 < stack 頂端高度，代表頂端柱子無法再向右延伸
  - 持續彈出並計算面積：`height * (i - startIndex)`
  - 記錄最後彈出元素的起始索引（當前柱子可以向左延伸到那個位置）
- 將 `{start, 當前高度}` 壓入 stack
- 遍歷完後，stack 中剩餘的柱子都可以延伸到最右端，繼續計算面積

時間複雜度：O(N)，空間複雜度：O(N)。

```cpp
class Solution {
public:
    int largestRectangleArea(vector<int>& heights) {
        int maxArea = 0;
        stack<pair<int,int>> st; // {startIndex, height}

        for (int i = 0; i < (int)heights.size(); i++) {
            int start = i;
            while (!st.empty() && st.top().second > heights[i]) {
                auto [idx, h] = st.top(); st.pop();
                maxArea = max(maxArea, h * (i - idx));
                start = idx; // 當前柱子可以延伸到這裡
            }
            st.push({start, heights[i]});
        }

        // 處理 stack 中剩餘的柱子（可延伸到最右端）
        while (!st.empty()) {
            auto [idx, h] = st.top(); st.pop();
            maxArea = max(maxArea, h * ((int)heights.size() - idx));
        }

        return maxArea;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N)
- Space Complexity: O(N)
