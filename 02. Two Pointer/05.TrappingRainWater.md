# [42. Trapping Rain Water](https://leetcode.com/problems/trapping-rain-water/)
Hard

Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it can trap after raining.

 

Example 1:
```
Input: height = [0,1,0,2,1,0,1,3,2,1,2,1]
Output: 6
Explanation: The above elevation map (black section) is represented by array [0,1,0,2,1,0,1,3,2,1,2,1]. 
In this case, 6 units of rain water (blue section) are being trapped.
```
Example 2:
```
Input: height = [4,2,0,3,2,5]
Output: 9
 ```

Constraints:

- n == height.length
- 1 <= n <= 2 * 10<sup>4</sup>
- 0 <= height[i] <= 10<sup>5</sup>

## 解題思路

### 方法一：前後綴最大值陣列
- 預先計算每個位置左邊最大高度（`leftMax`）與右邊最大高度（`rightMax`）
- 每個位置能儲水量 = `min(leftMax[i], rightMax[i]) - height[i]`（若為負則為 0）
- 時間 O(N)，空間 O(N)

### 方法二：雙指針（最佳）
- 用 `lMax`、`rMax` 兩個變數取代陣列
- 若 `lMax <= rMax`：處理左指針（左邊是瓶頸），更新 `lMax`，計算儲水量，左移
- 否則：處理右指針，更新 `rMax`，計算儲水量，右移
- 關鍵：當 `lMax <= rMax` 時，左邊的儲水量由 `lMax` 決定（右邊一定夠高）

時間複雜度：O(N)，空間複雜度：O(1)。

```cpp
class Solution {
public:
    int trap(vector<int>& height) {
        int left = 0, right = (int)height.size() - 1;
        int lMax = height[0], rMax = height[right];
        int ans = 0;

        while (left <= right) {
            if (lMax <= rMax) {
                lMax = max(lMax, height[left]);
                int val = min(lMax, rMax) - height[left];
                ans += max(0, val);
                left++;
            } else {
                rMax = max(rMax, height[right]);
                int val = min(lMax, rMax) - height[right];
                ans += max(0, val);
                right--;
            }
        }

        return ans;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N)
- Space Complexity: O(1)
