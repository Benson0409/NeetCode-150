# 15. 3Sum
Medium

Given an integer array nums, return all the triplets [nums[i], nums[j], nums[k]] such that i != j, i != k, and j != k, and nums[i] + nums[j] + nums[k] == 0.

Notice that the solution set must not contain duplicate triplets.

 

Example 1:
```
Input: nums = [-1,0,1,2,-1,-4]
Output: [[-1,-1,2],[-1,0,1]]
```
Example 2:
```
Input: nums = []
Output: []
```
Example 3:
```
Input: nums = [0]
Output: []
 ```

Constraints:

- 0 <= nums.length <= 3000
- -10<sup>5</sup> <= nums[i] <= 10<sup>5</sup>

## 解題思路

先將陣列排序，然後固定第一個數字 `a`，對剩餘部分用雙指針找出另外兩個數字 `b` 和 `c` 使三數之和為 0：

1. 排序後從第一個元素開始遍歷，跳過重複的元素（避免重複三元組）。
2. 對每個 `a`，用 `l = i+1`、`r = len(nums)-1` 做雙指針搜尋。
3. 找到答案後，移動 `l` 指針並跳過相同的元素以去重。

時間複雜度：O(n²)，空間複雜度：O(1) 或 O(n)（取決於排序算法）。

```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        res = []
        nums.sort()

        for i, a in enumerate(nums):
            # Skip positive numbers as they cannot sum to 0 with following positive numbers
            if a > 0:
                break

            if i > 0 and a == nums[i - 1]:
                continue

            l, r = i + 1, len(nums) - 1
            while l < r:
                threeSum = a + nums[l] + nums[r]
                if threeSum > 0:
                    r -= 1
                elif threeSum < 0:
                    l += 1
                else:
                    res.append([a, nums[l], nums[r]])
                    l += 1
                    r -= 1
                    while nums[l] == nums[l - 1] and l < r:
                        l += 1
        return res
```
