# 15. 3Sum
Medium

Given an integer array nums, return all the triplets [nums[i], nums[j], nums[k]] such that i != j, i != k, and j != k, and nums[i] + nums[j] + nums[k] == 0.

Notice that the solution set must not contain duplicate triplets.

 

Example 1:
```
Input: nums = [-1,0,1,2,-1,-4]
Output: [[-1,-1,2],[-1,0,1]]
```
Example 2:
```
Input: nums = []
Output: []
```
Example 3:
```
Input: nums = [0]
Output: []
 ```

Constraints:

- 0 <= nums.length <= 3000
- -10<sup>5</sup> <= nums[i] <= 10<sup>5</sup>

## 解題思路

先將陣列排序，然後固定第一個數字 `nums[i]`，對剩餘部分用雙指針找出另外兩個數字使三數之和為 0：

1. 排序後從 `i = 0` 遍歷到 `n-3`，跳過重複的 `nums[i]`（避免重複三元組）。
2. 對每個 `i`，用 `left = i+1`、`right = n-1` 做雙指針搜尋。
3. 找到答案後，跳過相同的 `left` 和 `right` 值（去重）。

時間複雜度：O(n²)，空間複雜度：O(1)（不計輸出）。

```cpp
class Solution {
    void search(int index, vector<int>& nums, vector<vector<int>>& result) {
        int left = index + 1, right = (int)nums.size() - 1;
        while (left < right) {
            int sum = nums[index] + nums[left] + nums[right];
            if (sum == 0) {
                result.push_back({nums[index], nums[left], nums[right]});
                left++;
                right--;
                while (left < right && nums[left] == nums[left - 1]) left++;
                while (left < right && nums[right] == nums[right + 1]) right--;
            } else if (sum < 0) {
                left++;
            } else {
                right--;
            }
        }
    }

public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> result;
        sort(nums.begin(), nums.end());
        for (int i = 0; i < (int)nums.size() - 2; i++) {
            if (i > 0 && nums[i] == nums[i - 1]) continue;
            search(i, nums, result);
        }
        return result;
    }
};
```
