# 19. Remove Nth Node From End of List
Medium

Given the head of a linked list, remove the nth node from the end of the list and return its head.

 

Example 1:
```
Input: head = [1,2,3,4,5], n = 2
Output: [1,2,3,5]
```
Example 2:
```
Input: head = [1], n = 1
Output: []
```
Example 3:
```
Input: head = [1,2], n = 1
Output: [1]
 ```

Constraints:

The number of nodes in the list is sz.
- 1 <= sz <= 30
- 1 <= n <= sz
 

### Follow up: Could you do this in one pass?

## 解題思路

使用雙指針（快慢指針）一次遍歷：

1. 建立 dummy node 指向 head（處理刪除頭節點的邊界情況）。
2. `right` 先走 n 步。
3. `left` 和 `right` 同步移動，直到 `right` 到達最後一個節點。
4. 此時 `left` 的下一個節點就是要刪除的節點：`left.next = left.next.next`。

時間複雜度：O(N)，空間複雜度：O(1)。

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        left = dummy
        right = head

        while n > 0:
            right = right.next
            n -= 1

        while right:
            left = left.next
            right = right.next

        # delete
        left.next = left.next.next
        return dummy.next
```

## Complexity Analysis
- Time Complexity: O(N)
- Space Complexity: O(1)
