# 19. Remove Nth Node From End of List
Medium

Given the head of a linked list, remove the nth node from the end of the list and return its head.

 

Example 1:
```
Input: head = [1,2,3,4,5], n = 2
Output: [1,2,3,5]
```
Example 2:
```
Input: head = [1], n = 1
Output: []
```
Example 3:
```
Input: head = [1,2], n = 1
Output: [1]
 ```

Constraints:

The number of nodes in the list is sz.
- 1 <= sz <= 30
- 0 <= Node.val <= 100
- 1 <= n <= sz
 

### Follow up: Could you do this in one pass?

## 解題思路

使用雙指針（快慢指針）一次遍歷：

1. 建立 dummy node 指向 head（處理刪除頭節點的邊界情況）
2. `fast` 先走 n 步
3. `slow` 和 `fast` 同步移動，直到 `fast` 到達最後一個節點
4. 此時 `slow` 的下一個節點就是要刪除的節點：`slow->next = slow->next->next`

時間複雜度：O(N)，空間複雜度：O(1)。

```cpp
class Solution {
public:
    ListNode* removeNthFromEnd(ListNode* head, int n) {
        ListNode dummy(0);
        dummy.next = head;
        ListNode* slow = &dummy, *fast = &dummy;

        // fast 先走 n 步
        for (int i = 0; i < n; i++)
            fast = fast->next;

        // 同步移動直到 fast 到達最後一個節點
        while (fast->next) {
            slow = slow->next;
            fast = fast->next;
        }

        slow->next = slow->next->next;

        return dummy.next;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N)（一次遍歷）
- Space Complexity: O(1)
