# 217. Contains Duplicate
Easy

Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.

 

Example 1:
```
Input: nums = [1,2,3,1]
Output: true
```
Example 2:
```
Input: nums = [1,2,3,4]
Output: false
```
Example 3:
```
Input: nums = [1,1,1,3,3,4,3,2,4,2]
Output: true
 ```

Constraints:

- 1 <= nums.length <= 105
- -109 <= nums[i] <= 109

## 解題思路

用一個 HashSet（`unordered_set`）記錄已經遍歷過的數字。每次遇到新數字時，先檢查是否已存在 set 中：
- 若存在 → 代表有重複，直接回傳 `true`
- 若不存在 → 加入 set，繼續遍歷

時間複雜度：O(n)，空間複雜度：O(n)。

```cpp
class Solution {
public:
    bool containsDuplicate(vector<int>& nums) {
        unordered_set<int> seen;
        for (int num : nums) {
            if (!seen.insert(num).second)
                return true;
        }
        return false;
    }
};
```
