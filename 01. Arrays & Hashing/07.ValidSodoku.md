# 36. Valid Sudoku
Medium

Determine if a 9 x 9 Sudoku board is valid. Only the filled cells need to be validated according to the following rules:

- Each row must contain the digits 1-9 without repetition.
- Each column must contain the digits 1-9 without repetition.
- Each of the nine 3 x 3 sub-boxes of the grid must contain the digits 1-9 without repetition.

Note:
- A Sudoku board (partially filled) could be valid but is not necessarily solvable.
- Only the filled cells need to be validated according to the mentioned rules.
 

Example 1:

```
Input: board = 
[["5","3",".",".","7",".",".",".","."]
,["6",".",".","1","9","5",".",".","."]
,[".","9","8",".",".",".",".","6","."]
,["8",".",".",".","6",".",".",".","3"]
,["4",".",".","8",".","3",".",".","1"]
,["7",".",".",".","2",".",".",".","6"]
,[".","6",".",".",".",".","2","8","."]
,[".",".",".","4","1","9",".",".","5"]
,[".",".",".",".","8",".",".","7","9"]]
Output: true
```
Example 2:
```
Input: board = 
[["8","3",".",".","7",".",".",".","."]
,["6",".",".","1","9","5",".",".","."]
,[".","9","8",".",".",".",".","6","."]
,["8",".",".",".","6",".",".",".","3"]
,["4",".",".","8",".","3",".",".","1"]
,["7",".",".",".","2",".",".",".","6"]
,[".","6",".",".",".",".","2","8","."]
,[".",".",".","4","1","9",".",".","5"]
,[".",".",".",".","8",".",".","7","9"]]
Output: false
```
Explanation: Same as Example 1, except with the 5 in the top left corner being modified to 8. Since there are two 8's in the top left 3x3 sub-box, it is invalid.
 

Constraints:

- board.length == 9
- board[i].length == 9
- board[i][j] is a digit 1-9 or '.'.

## 解題思路

**方法一：逐格驗證（自訂輔助函式）**

對每個非空格（`.`）的格子，分別檢查它所在的列（row）、行（col）、以及 3×3 小方格（box）中是否有重複。

**方法二：使用 Set 一次掃描**

用一個 `unordered_set<string>` 記錄所有已見過的「數字 + 位置」資訊（如 `"5 in row 2"`、`"5 in col 3"`、`"5 in block 0-1"`）。若插入失敗則代表重複，直接回傳 `false`。

```cpp
// 方法一：逐格驗證
class Solution {
    bool isValidRow(char c, int row, int col, vector<vector<char>>& board) {
        for (int i = 0; i < 9; i++)
            if (board[row][i] == c && i != col) return false;
        return true;
    }

    bool isValidCol(char c, int row, int col, vector<vector<char>>& board) {
        for (int i = 0; i < 9; i++)
            if (board[i][col] == c && i != row) return false;
        return true;
    }

    bool isValidGrid(char c, int row, int col, vector<vector<char>>& board) {
        int r0 = 3 * (row / 3), c0 = 3 * (col / 3);
        for (int i = r0; i < r0 + 3; i++)
            for (int j = c0; j < c0 + 3; j++)
                if (board[i][j] == c && i != row && j != col) return false;
        return true;
    }

public:
    bool isValidSudoku(vector<vector<char>>& board) {
        for (int i = 0; i < 9; i++) {
            for (int j = 0; j < 9; j++) {
                char c = board[i][j];
                if (c != '.' && (!isValidRow(c, i, j, board) ||
                                 !isValidCol(c, i, j, board) ||
                                 !isValidGrid(c, i, j, board)))
                    return false;
            }
        }
        return true;
    }
};
```

```cpp
// 方法二：使用 Set（更簡潔）
class Solution {
public:
    bool isValidSudoku(vector<vector<char>>& board) {
        unordered_set<string> seen;
        for (int i = 0; i < 9; i++) {
            for (int j = 0; j < 9; j++) {
                char c = board[i][j];
                if (c == '.') continue;
                string num(1, c);
                if (!seen.insert(num + " row " + to_string(i)).second ||
                    !seen.insert(num + " col " + to_string(j)).second ||
                    !seen.insert(num + " block " + to_string(i/3) + "-" + to_string(j/3)).second)
                    return false;
            }
        }
        return true;
    }
};
```
