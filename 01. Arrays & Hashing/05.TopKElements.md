# 347. Top K Frequent Elements
Medium

Given an integer array nums and an integer k, return the k most frequent elements. You may return the answer in any order.

 

Example 1:
```
Input: nums = [1,1,1,2,2,3], k = 2
Output: [1,2]
```
Example 2:
```
Input: nums = [1], k = 1
Output: [1]
 ```

Constraints:

- 1 <= nums.length <= 105
- k is in the range [1, the number of unique elements in the array].
- It is guaranteed that the answer is unique.

### Follow up: Your algorithm's time complexity must be better than O(n log n), where n is the array's size.

## 解題思路

1. 用 `unordered_map` 統計每個數字出現的頻率。
2. 用 max-heap（`priority_queue`）依頻率排序，取出頻率最高的前 k 個元素。

C++ 的 `priority_queue` 預設是 max-heap，直接用 `{freq, num}` 的 pair 作為元素即可依頻率降序排列。

時間複雜度：O(n log n)（heap 操作）；若需嚴格 O(n)，可改用 bucket sort。

```cpp
class Solution {
public:
    vector<int> topKFrequent(vector<int>& nums, int k) {
        unordered_map<int, int> freq;
        for (int num : nums)
            freq[num]++;

        // max-heap: {frequency, number}
        priority_queue<pair<int,int>> pq;
        for (auto& [num, cnt] : freq)
            pq.push({cnt, num});

        vector<int> res;
        for (int i = 0; i < k; i++) {
            res.push_back(pq.top().second);
            pq.pop();
        }
        return res;
    }
};
```
