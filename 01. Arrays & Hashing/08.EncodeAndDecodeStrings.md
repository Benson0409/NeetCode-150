# 659 · Encode and Decode Strings

Description
Design an algorithm to encode a list of strings to a string. The encoded string is then sent over the network and is decoded back to the original list of strings.

Please implement encode and decode


Example1
```
Input: ["lint","code","love","you"]
Output: ["lint","code","love","you"]
Explanation:
One possible encode method is: "lint:;code:;love:;you"
```
Example2
```
Input: ["we", "say", ":", "yes"]
Output: ["we", "say", ":", "yes"]
Explanation:
One possible encode method is: "we:;say:;:::;yes"
```

## 解題思路

關鍵問題：如何避免分隔符號與字串內容衝突？

使用「長度前綴法（length-prefix encoding）」：

- **Encode**：對每個字串，先寫入其長度，再加上一個分隔符（如 `` ` ``），最後接字串本身。例如 `"hello"` → `"5`hello"`。
- **Decode**：讀取長度數字直到分隔符，然後取出對應長度的子字串，不需擔心字串內容包含任何特殊符號。

這樣任意字串（包含特殊符號）都能正確編解碼。

```cpp
class Solution {
public:
    string encode(vector<string>& strs) {
        string res;
        for (const string& s : strs) {
            res += to_string(s.size()) + '`' + s;
        }
        return res;
    }

    vector<string> decode(string str) {
        vector<string> strs;
        int i = 0;
        while (i < (int)str.size()) {
            int j = i;
            while (str[j] != '`') j++;
            int len = stoi(str.substr(i, j - i));
            strs.push_back(str.substr(j + 1, len));
            i = j + 1 + len;
        }
        return strs;
    }
};
```
