# [124. Binary Tree Maximum Path Sum](https://leetcode.com/problems/binary-tree-maximum-path-sum/)
Hard

A path in a binary tree is a sequence of nodes where each pair of adjacent nodes in the sequence has an edge connecting them. A node can only appear in the sequence at most once. Note that the path does not need to pass through the root.

The path sum of a path is the sum of the node's values in the path.

Given the root of a binary tree, return the maximum path sum of any non-empty path.

 

Example 1:
```
Input: root = [1,2,3]
Output: 6
Explanation: The optimal path is 2 -> 1 -> 3 with a path sum of 2 + 1 + 3 = 6.
```
Example 2:
```
Input: root = [-10,9,20,null,null,15,7]
Output: 42
Explanation: The optimal path is 15 -> 20 -> 7 with a path sum of 15 + 20 + 7 = 42.
 ```

Constraints:
- The number of nodes in the tree is in the range [1, 3 * 104].
- -1000 <= Node.val <= 1000

## 解題思路

對每個節點，考慮以該節點為「轉折點」（最高點）的路徑：

- 左子樹貢獻 = `max(0, helper(left))`（若為負則不取）
- 右子樹貢獻 = `max(0, helper(right))`
- 穿越該節點的最大路徑 = `left + val + right` → 更新全域最大值
- 函式回傳給父節點的值：只能選一個方向延伸 = `val + max(left, right)`

注意：路徑可以只包含單一節點（值可能為負），所以全域最大值初始化為 `INT_MIN`。

時間複雜度：O(N)，空間複雜度：O(H)。

```cpp
class Solution {
    int maxSum;

    int helper(TreeNode* root) {
        if (!root) return 0;
        int val = root->val;
        int lVal = max(0, helper(root->left));
        int rVal = max(0, helper(root->right));
        maxSum = max(maxSum, lVal + val + rVal);
        return val + max(lVal, rVal);
    }

public:
    int maxPathSum(TreeNode* root) {
        maxSum = INT_MIN;
        helper(root);
        return maxSum;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N)
- Space Complexity: O(H)，H 為樹高
