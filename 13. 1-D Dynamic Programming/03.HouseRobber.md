# 198. House Robber
Medium

You are given an integer array nums representing the amount of money of each house. Return the maximum amount of money you can rob tonight without alerting the police (cannot rob adjacent houses).

 

Example 1:
```
Input: nums = [1,2,3,1]
Output: 4
```

Constraints:
- 1 <= nums.length <= 100

## 解題思路

動態規劃：對每棟房子決定「搶」或「不搶」。

1. 假設 `rob1` 是搶到前兩間房子的最大值，`rob2` 是搶到前一間房子的最大值。
2. 對於當前房子 `n`：
   - 選擇搶：則最大值為 `n + rob1`。
   - 選擇不搶：則最大值為 `rob2`。
3. 更新 `temp = max(n + rob1, rob2)`，然後移動窗口：`rob1 = rob2`, `rob2 = temp`。

時間複雜度：O(N)，空間複雜度：O(1)。

```python
class Solution:
    def rob(self, nums: List[int]) -> int:
        rob1, rob2 = 0, 0

        for n in nums:
            temp = max(n + rob1, rob2)
            rob1 = rob2
            rob2 = temp
        return rob2
```

## Complexity Analysis
- Time Complexity: O(n)
- Space Complexity: O(1)
