# 152. Maximum Product Subarray
Medium

Given an integer array nums, find a contiguous non-empty subarray within the array that has the largest product, and return the product.

 

Example 1:
```
Input: nums = [2,3,-2,4]
Output: 6
```

Constraints:
- 1 <= nums.length <= 2 × 10^4

## 解題思路

動態規劃：同時追蹤當前子陣列的最大積與最小積。

1. 因為負負得正，當前的負數乘以之前的最小積（如果是負數）可能會變成最大積。
2. 對於每個數字 `n`：
   - 計算 `n * curMax`, `n * curMin`, 以及 `n` 本身的最大與最小值。
   - 更新 `curMax` 與 `curMin`。
   - 特別注意：若 `n` 為 0，則 `curMax` 與 `curMin` 都會被重置。
3. 在遍歷過程中持續更新全域最大積 `res`。

時間複雜度：O(N)，空間複雜度：O(1)。

```python
class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        res = max(nums)
        curMin, curMax = 1, 1

        for n in nums:
            tmp = curMax * n
            curMax = max(n * curMax, n * curMin, n)
            curMin = min(tmp, n * curMin, n)
            res = max(res, curMax)
        return res
```

## Complexity Analysis
- Time Complexity: O(n)
- Space Complexity: O(1)
