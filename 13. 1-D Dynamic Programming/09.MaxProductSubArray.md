# [152. Maximum Product Subarray](https://leetcode.com/problems/maximum-product-subarray/)
Medium

Given an integer array nums, find a contiguous non-empty subarray within the array that has the largest product, and return the product.

 

Example 1:
```
Input: nums = [2,3,-2,4]
Output: 6
```
Example 2:
```
Input: nums = [-2,0,-1]
Output: 0
 ```

Constraints:

- 1 <= nums.length <= 2 × 10^4
- -10 <= nums[i] <= 10

## 解題思路

同時追蹤當前最大值與最小值：

- 遇到負數時，最大值與最小值會互換（負 × 負 = 正）
- 遇到 `0` 時重置為 1（乘以 0 會使後續乘積歸零）
- `curMin = min(prevMin × nums[i], prevMax × nums[i], nums[i])`
- `curMax = max(prevMin × nums[i], prevMax × nums[i], nums[i])`

時間複雜度：O(N)，空間複雜度：O(1)。

```cpp
class Solution {
public:
    int maxProduct(vector<int>& nums) {
        int curMin = nums[0], curMax = nums[0];
        int result = nums[0];

        for (int i = 1; i < (int)nums.size(); i++) {
            int a = nums[i];
            int tmpMin = min({curMin * a, curMax * a, a});
            int tmpMax = max({curMin * a, curMax * a, a});
            curMin = tmpMin;
            curMax = tmpMax;
            result = max(result, curMax);
        }
        return result;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N)
- Space Complexity: O(1)
