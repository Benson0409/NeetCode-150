# 746. Min Cost Climbing Stairs
Easy

You are given an integer array cost where cost[i] is the cost of ith step on a staircase. Once you pay the cost, you can either climb one or two steps. You can either start from the step with index 0, or the step with index 1.

Return the minimum cost to reach the top of the floor.

 

Example 1:
```
Input: cost = [10,15,20]
Output: 15
```

Constraints:
- 2 <= cost.length <= 1000

## 解題思路

動態規劃（從後往前）：

1. 目標是到達 `len(cost)` 這一階。
2. 對於每一階 `i`，到達頂端的最小成本是 `cost[i] + min(到達頂端的剩餘成本1, 到達頂端的剩餘成本2)`。
3. 我們可以從倒數第三階開始向前遍歷，將 `cost[i]` 更新為到達頂端的最小累積成本。
4. 最後回傳 `min(cost[0], cost[1])`。

時間複雜度：O(N)，空間複雜度：O(1)。

```python
class Solution:
    def minCostClimbingStairs(self, cost: List[int]) -> int:
        for i in range(len(cost) - 3, -1, -1):
            cost[i] += min(cost[i + 1], cost[i + 2])

        return min(cost[0], cost[1])
```

## Complexity Analysis
- Time Complexity: O(n)
- Space Complexity: O(1)
