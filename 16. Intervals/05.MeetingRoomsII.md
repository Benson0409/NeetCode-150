# [919 · Meeting Rooms II](https://www.lintcode.com/problem/919/)
Medium

Given an array of meeting time intervals consisting of start and end times [[s1,e1],[s2,e2],...] (si < ei), find the minimum number of conference rooms required.

> (0,8),(8,10) is not conflict at 8

Example 1
```
Input: intervals = [(0,30),(5,10),(15,20)]
Output: 2
```
Example 2
```
Input: intervals = [(2,7)]
Output: 1
```

## 解題思路

雙指標法（分別排序 start 與 end）：

1. 將所有起始時間與結束時間分開排序
2. 用 `s` 指向起始、`e` 指向結束；若 `start[s] < end[e]`，代表新會議開始時前一個未結束 → 需要新房間（count++, s++）
3. 否則一間房間釋放（count--, e++）
4. 全程追蹤 `usedRooms = max(usedRooms, count)`

時間複雜度：O(N log N)，空間複雜度：O(N)。

```cpp
class Solution {
public:
    int minMeetingRooms(vector<vector<int>>& intervals) {
        int n = intervals.size();
        if (n == 0) return 0;

        vector<int> starts(n), ends(n);
        for (int i = 0; i < n; i++) {
            starts[i] = intervals[i][0];
            ends[i] = intervals[i][1];
        }
        sort(starts.begin(), starts.end());
        sort(ends.begin(), ends.end());

        int s = 0, e = 0, count = 0, rooms = 0;
        while (s < n) {
            if (starts[s] < ends[e]) { count++; s++; }
            else { count--; e++; }
            rooms = max(rooms, count);
        }
        return rooms;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N log N)
- Space Complexity: O(N)
