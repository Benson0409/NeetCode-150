# 57. Insert Interval
Medium

You are given an array of non-overlapping intervals sorted in ascending order by start time. You are also given a `newInterval = [start, end]`. Insert `newInterval` into `intervals` such that `intervals` is still sorted and non-overlapping (merge if necessary).

 

Example 1:
```
Input: intervals = [[1,3],[6,9]], newInterval = [2,5]
Output: [[1,5],[6,9]]
```

Constraints:
- 0 <= intervals.length <= 10^4
- intervals is sorted by start_i in ascending order.

## 解題思路

三段式處理法：

1. **左側不重疊區間**：
   - 遍歷 `intervals`，將所有結束時間 `intervals[i][1]` 小於新區間起始時間 `newInterval[0]` 的區間直接加入結果列表。
2. **合併重疊區間**：
   - 只要當前區間的起始時間 `intervals[i][0]` 小於等於新區間的結束時間 `newInterval[1]`，就代表有重疊。
   - 合併方法：更新新區間的 `start = min(start, intervals[i][0])`，`end = max(end, intervals[i][1])`。
   - 將最終合併後的新區間加入結果列表。
3. **右側不重疊區間**：
   - 將剩餘的所有區間加入結果列表。

時間複雜度：O(N)，空間複雜度：O(N)。

```python
class Solution:
    def insert(self, intervals: List[List[int]], newInterval: List[int]) -> List[List[int]]:
        res = []

        for i in range(len(intervals)):
            if newInterval[1] < intervals[i][0]:
                res.append(newInterval)
                return res + intervals[i:]
            elif newInterval[0] > intervals[i][1]:
                res.append(intervals[i])
            else:
                newInterval = [
                    min(newInterval[0], intervals[i][0]),
                    max(newInterval[1], intervals[i][1]),
                ]
        res.append(newInterval)
        return res
```

## Complexity Analysis
- Time Complexity: O(n)
- Space Complexity: O(n)
