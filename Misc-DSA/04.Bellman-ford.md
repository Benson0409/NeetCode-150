# Bellman-Ford Algorithm

Bellman-Ford 是單源最短路徑（SSSP）演算法，可處理**負權邊**，並能偵測**負環**。相較於 Dijkstra（不支援負權），時間複雜度較高 O(V × E)，但適用場景更廣。

## 解題思路

1. 初始化 `dist[src] = 0`，其餘為 `∞`
2. 對所有邊進行 V-1 次鬆弛（relax）
3. 再跑一輪，若仍能鬆弛，代表存在負環（該節點距離設為 `-∞`）

時間複雜度：O(V × E)，空間複雜度：O(V)。

```cpp
struct Edge {
    int from, to;
    double cost;
};

vector<double> bellmanFord(vector<Edge>& edges, int V, int start) {
    vector<double> dist(V, 1e18);
    dist[start] = 0;

    bool relaxed = true;
    for (int v = 0; v < V - 1 && relaxed; v++) {
        relaxed = false;
        for (auto& e : edges) {
            if (dist[e.from] + e.cost < dist[e.to]) {
                dist[e.to] = dist[e.from] + e.cost;
                relaxed = true;
            }
        }
    }

    // 偵測負環
    relaxed = true;
    for (int v = 0; v < V - 1 && relaxed; v++) {
        relaxed = false;
        for (auto& e : edges) {
            if (dist[e.from] + e.cost < dist[e.to]) {
                dist[e.to] = -1e18; // 負環節點
                relaxed = true;
            }
        }
    }
    return dist;
}
```

## Complexity Analysis
- Time Complexity: O(V × E)
- Space Complexity: O(V)
