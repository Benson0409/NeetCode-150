# Bellman-Ford Algorithm

## 解題思路

Bellman-Ford 演算法用於在包含負權邊的圖中尋找單源最短路徑，並可偵測負環。

1. **核心**：對所有邊進行 `V-1` 次鬆弛 (Relaxation) 操作。
2. **流程**：
   - 初始化起點距離為 0，其餘為 `infinity`。
   - 重複 `V-1` 次遍歷所有邊：
     - 若 `dist[u] + weight < dist[v]`，更新 `dist[v]`。
   - **負環偵測**：再遍歷一次所有邊，若仍有邊可被鬆弛，則存在負環。

時間複雜度：O(V × E)，空間複雜度：O(V)。

```python
def bellman_ford(edges, V, start):
    # edges: list of (u, v, weight)
    # V: number of vertices
    dist = [float("inf")] * V
    dist[start] = 0

    # Relax all edges V - 1 times
    for _ in range(V - 1):
        for u, v, w in edges:
            if dist[u] != float("inf") and dist[u] + w < dist[v]:
                dist[v] = dist[u] + w

    # Check for negative cycles
    for u, v, w in edges:
        if dist[u] != float("inf") and dist[u] + w < dist[v]:
            print("Graph contains negative weight cycle")
            return None

    return dist
```
