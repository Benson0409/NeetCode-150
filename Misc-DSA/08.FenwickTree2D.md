# 2D Fenwick Tree (2D Binary Indexed Tree)

二維 Fenwick Tree 將一維 BIT 擴展到二維，支援二維前綴和查詢與單點更新。

利用容斥原理計算矩形區域的區間和：
`rangeSum(x1,y1,x2,y2) = query(x2,y2) - query(x1-1,y2) - query(x2,y1-1) + query(x1-1,y1-1)`

時間複雜度：update/query O(log M × log N)；空間複雜度：O(M × N)。

```cpp
class FenwickTree2D {
    vector<vector<int>> ft;
    int rows, cols;

    int lsOne(int i) { return i & (-i); }

public:
    FenwickTree2D(vector<vector<int>>& arr)
        : rows(arr.size()), cols(arr[0].size()),
          ft(arr.size() + 1, vector<int>(arr[0].size() + 1, 0)) {
        for (int i = 0; i < rows; i++)
            for (int j = 0; j < cols; j++)
                update(i + 1, j + 1, arr[i][j]);
    }

    void update(int row, int col, int val) {  // 1-indexed
        for (int i = row; i <= rows; i += lsOne(i))
            for (int j = col; j <= cols; j += lsOne(j))
                ft[i][j] += val;
    }

    int querySum(int row, int col) {  // prefix sum [1,1] to [row,col]
        int sum = 0;
        for (int i = row; i > 0; i -= lsOne(i))
            for (int j = col; j > 0; j -= lsOne(j))
                sum += ft[i][j];
        return sum;
    }

    // 0-indexed rectangle [x1,y1] to [x2,y2]
    int rangeSum(int x1, int y1, int x2, int y2) {
        return querySum(x2+1, y2+1) - querySum(x1, y2+1)
             - querySum(x2+1, y1) + querySum(x1, y1);
    }
};
```

## Complexity Analysis
- Time Complexity: update/query O(log M × log N)
- Space Complexity: O(M × N)
