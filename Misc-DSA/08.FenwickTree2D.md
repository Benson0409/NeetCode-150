# 2D Fenwick Tree (2D Binary Indexed Tree)

## 解題思路

二維 Fenwick Tree 是 BIT 的二維擴充，支援矩形區域內的前綴和查詢與單點更新。

1. **更新**：對於點 `(r, c)`，在兩個維度上分別沿著 `i += i & (-i)` 向上更新。
2. **查詢**：對於點 `(r, c)`，累加 `(r, c)` 到 `(1, 1)` 的所有相關節點。
3. **區域和**：利用容斥原理（Inclusion-Exclusion Principle）：
   - `Sum(r1, c1, r2, c2) = Total(r2, c2) - Total(r1-1, c2) - Total(r2, c1-1) + Total(r1-1, c1-1)`。

時間複雜度：O(log M × log N)，空間複雜度：O(M × N)。

```python
class FenwickTree2D:
    def __init__(self, rows, cols):
        self.rows = rows
        self.cols = cols
        self.tree = [[0] * (cols + 1) for _ in range(rows + 1)]

    def update(self, r, c, delta):
        # 1-indexed
        i = r
        while i <= self.rows:
            j = c
            while j <= self.cols:
                self.tree[i][j] += delta
                j += j & (-j)
            i += i & (-i)

    def query(self, r, c):
        # 1-indexed, returns sum from (1,1) to (r,c)
        sum_ = 0
        i = r
        while i > 0:
            j = c
            while j > 0:
                sum_ += self.tree[i][j]
                j -= j & (-j)
            i -= i & (-i)
        return sum_

    def range_query(self, r1, c1, r2, c2):
        # 1-indexed, returns rectangle sum
        return (self.query(r2, c2) 
                - self.query(r1 - 1, r2) 
                - self.query(r2, c1 - 1) 
                + self.query(r1 - 1, c1 - 1))
```
