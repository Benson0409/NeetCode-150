# Union-Find (Disjoint Set Union)

## 解題思路

并查集用於快速判斷兩個元素是否屬於同一集合，並合併集合。

兩個關鍵操作：
- `find(x)`：找根節點（路徑壓縮）
- `union(x, y)`：按 rank 合併兩棵樹

時間複雜度：幾乎 O(1)（α(N) 反阿克曼函數），空間複雜度：O(N)。

```cpp
class UnionFind {
    vector<int> root, rank_;

public:
    UnionFind(int size) : root(size), rank_(size, 0) {
        for (int i = 0; i < size; i++) root[i] = i;
    }

    int find(int x) {
        if (x != root[x]) root[x] = find(root[x]); // 路徑壓縮
        return root[x];
    }

    void unite(int x, int y) {
        int rx = find(x), ry = find(y);
        if (rx == ry) return;
        if (rank_[rx] > rank_[ry]) root[ry] = rx;
        else if (rank_[rx] < rank_[ry]) root[rx] = ry;
        else { root[ry] = rx; rank_[rx]++; }
    }

    bool isConnected(int x, int y) {
        return find(x) == find(y);
    }
};
```
