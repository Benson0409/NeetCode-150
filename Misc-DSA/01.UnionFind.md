# Union-Find (Disjoint Set Union)

## 解題思路

并查集（Union-Find）用於快速判斷兩個元素是否屬於同一集合，並合併兩個不相交的集合。

兩個關鍵優化：
1. **路徑壓縮 (Path Compression)**：在 `find` 時，將路徑上的所有節點直接指向根節點，使後續查詢達到近似 O(1)。
2. **按秩合併 (Union by Rank/Size)**：將較矮的樹合併到較高的樹下，保持樹的平衡，避免退化成鏈結串列。

時間複雜度：幾乎 O(1)（反阿克曼函數 α(N)），空間複雜度：O(N)。

```python
class UnionFind:
    def __init__(self, n):
        self.parent = [i for i in range(n)]
        self.rank = [1] * n

    def find(self, i):
        # Path compression
        if self.parent[i] == i:
            return i
        self.parent[i] = self.find(self.parent[i])
        return self.parent[i]

    def union(self, i, j):
        root_i = self.find(i)
        root_j = self.find(j)
        if root_i != root_j:
            # Union by rank
            if self.rank[root_i] > self.rank[root_j]:
                self.parent[root_j] = root_i
            elif self.rank[root_i] < self.rank[root_j]:
                self.parent[root_i] = root_j
            else:
                self.parent[root_i] = root_j
                self.rank[root_j] += 1
            return True
        return False

    def is_connected(self, i, j):
        return self.find(i) == self.find(j)
```
