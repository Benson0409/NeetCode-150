# KMP String Matching Algorithm

## [28. Find the Index of the First Occurrence in a String](https://leetcode.com/problems/find-the-index-of-the-first-occurrence-in-a-string/)
Easy

Given two strings needle and haystack, return the index of the first occurrence of needle in haystack, or -1 if needle is not part of haystack.

Example 1:
```
Input: haystack = "hello", needle = "ll"
Output: 2
```
Example 2:
```
Input: haystack = "aaaaa", needle = "bba"
Output: -1
```

## 解題思路

KMP 演算法分兩步：

1. **建立 failure function（LPS 陣列）**：`lps[i]` 為 `needle[0..i]` 的最長相同前後綴長度，用於在不匹配時跳過已知可以跳過的比較
2. **匹配**：`i` 指向 haystack，`j` 指向 needle；匹配時同步移動，不匹配時將 `j` 跳回 `lps[j-1]`

時間複雜度：O(N + M)，N = haystack 長度，M = needle 長度；空間複雜度：O(M)。

```cpp
class Solution {
    vector<int> buildLPS(const string& needle) {
        int m = needle.size();
        vector<int> lps(m, -1);
        int j = 0, i = 1;
        while (i < m) {
            if (needle[i] == needle[j]) {
                lps[i++] = j++;
            } else if (j > 0) {
                j = lps[j-1] + 1;
            } else {
                i++;
            }
        }
        return lps;
    }

public:
    int strStr(string haystack, string needle) {
        if (needle.empty()) return 0;
        vector<int> lps = buildLPS(needle);
        int n = haystack.size(), m = needle.size();
        int i = 0, j = 0;
        while (i + m - j <= n) {
            if (haystack[i] == needle[j]) {
                if (j == m - 1) return i - j;
                i++; j++;
            } else if (j > 0) {
                j = lps[j-1] + 1;
            } else {
                i++;
            }
        }
        return -1;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N + M)
- Space Complexity: O(M)
