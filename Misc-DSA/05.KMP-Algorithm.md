# KMP String Matching Algorithm

## 解題思路

KMP 演算法分兩步：

1. **建立 LPS 陣列 (Longest Prefix Suffix)**：
   - `lps[i]` 儲存子字串 `pattern[0...i]` 中，同時是前綴也是後綴的最長長度。
   - 用於在不匹配時，決定模式字串應該跳轉到的位置，避免從頭比較。
2. **搜尋過程**：
   - 遍歷 `haystack`，若字元匹配則繼續。
   - 若不匹配且 `j > 0`，將 `j` 更新為 `lps[j - 1]`。

時間複雜度：O(N + M)，空間複雜度：O(M)。

```python
class Solution:
    def strStr(self, haystack: str, needle: str) -> int:
        if not needle: return 0
        
        # 1. Build LPS array
        lps = [0] * len(needle)
        prevLPS, i = 0, 1
        while i < len(needle):
            if needle[i] == needle[prevLPS]:
                lps[i] = prevLPS + 1
                prevLPS += 1
                i += 1
            elif prevLPS == 0:
                lps[i] = 0
                i += 1
            else:
                prevLPS = lps[prevLPS - 1]

        # 2. Match
        i = 0 # ptr for haystack
        j = 0 # ptr for needle
        while i < len(haystack):
            if haystack[i] == needle[j]:
                i, j = i + 1, j + 1
            else:
                if j == 0:
                    i += 1
                else:
                    j = lps[j - 1]
            if j == len(needle):
                return i - len(needle)
        
        return -1
```
