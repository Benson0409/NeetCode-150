# Segment Tree

線段樹用於高效處理區間查詢（如區間最小值、最大值、加總）和單點更新。以完全二元樹的陣列形式儲存。

## 解題思路

- `build`：遞迴建樹，葉節點儲存原陣列的索引，內部節點儲存左右子樹中值最小的索引（RMQ）
- `rmq`：區間最小值查詢，O(log N)
- `update`：單點更新，O(log N)

時間複雜度：build O(N)，query/update O(log N)；空間複雜度：O(N)。

```cpp
class SegmentTree {
    vector<int> st;
    vector<int> A;
    int n;

    int left(int p) { return p << 1; }
    int right(int p) { return (p << 1) | 1; }

    void build(int p, int L, int R) {
        if (L == R) { st[p] = L; return; }
        int mid = (L + R) / 2;
        build(left(p), L, mid);
        build(right(p), mid + 1, R);
        st[p] = (A[st[left(p)]] <= A[st[right(p)]]) ? st[left(p)] : st[right(p)];
    }

    int rmq(int p, int L, int R, int i, int j) {
        if (i > R || j < L) return -1;
        if (L >= i && R <= j) return st[p];
        int mid = (L + R) / 2;
        int p1 = rmq(left(p), L, mid, i, j);
        int p2 = rmq(right(p), mid + 1, R, i, j);
        if (p1 == -1) return p2;
        if (p2 == -1) return p1;
        return (A[p1] <= A[p2]) ? p1 : p2;
    }

    int update(int p, int L, int R, int idx, int val) {
        if (idx > R || idx < L) return st[p];
        if (L == R) { A[idx] = val; return st[p] = L; }
        int mid = (L + R) / 2;
        int p1 = update(left(p), L, mid, idx, val);
        int p2 = update(right(p), mid + 1, R, idx, val);
        return st[p] = (A[p1] <= A[p2]) ? p1 : p2;
    }

public:
    SegmentTree(vector<int>& arr) : A(arr), n(arr.size()), st(4 * arr.size()) {
        build(1, 0, n - 1);
    }

    int query(int i, int j) { return rmq(1, 0, n - 1, i, j); }
    void update(int idx, int val) { update(1, 0, n - 1, idx, val); }
};
```

## Complexity Analysis
- Time Complexity: build O(N), query/update O(log N)
- Space Complexity: O(N)
