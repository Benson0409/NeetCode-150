# BFS on Graph

BFS 將頂點分為三種狀態：
- **Undiscovered**：初始未訪問狀態
- **Discovered**：已發現但鄰居尚未全部處理
- **Processed**：所有出邊已處理完畢

## 解題思路

從起始節點加入佇列，逐層展開：每次取出一個頂點，處理其所有鄰居（未發現的加入佇列），直到佇列為空。

時間複雜度：O(V + E)，空間複雜度：O(V)。

```cpp
#include <vector>
#include <queue>
#include <iostream>
using namespace std;

class Graph {
public:
    int V;
    bool directed;
    vector<vector<pair<int,int>>> adjList; // {neighbor, weight}

    Graph(int v, bool dir = false) : V(v), directed(dir), adjList(v) {}

    void insertEdge(int x, int y, int weight = 0) {
        adjList[x].push_back({y, weight});
        if (!directed) adjList[y].push_back({x, weight});
    }
};

class BfsGraph {
    vector<bool> processed, discovered;
    vector<int> parent;
    Graph& g;

public:
    BfsGraph(Graph& g) : g(g),
        processed(g.V, false),
        discovered(g.V, false),
        parent(g.V, -1) {}

    void bfs(int start) {
        queue<int> q;
        q.push(start);
        discovered[start] = true;

        while (!q.empty()) {
            int v = q.front(); q.pop();
            // process vertex early
            cout << v << "\n";
            processed[v] = true;

            for (auto& [y, w] : g.adjList[v]) {
                if (!processed[y] || g.directed)
                    cout << v << "-" << y << "\n";
                if (!discovered[y]) {
                    q.push(y);
                    discovered[y] = true;
                    parent[y] = v;
                }
            }
            // process vertex late
        }
    }

    void findPath(int start, int end) {
        if (start == end || end == -1)
            cout << start << "\n";
        else {
            findPath(start, parent[end]);
            cout << end << "\n";
        }
    }
};
```

## Complexity Analysis
- Time Complexity: O(V + E)
- Space Complexity: O(V)
