# BFS on Graph

## 解題思路

廣度優先搜尋 (BFS) 常用於尋找無權圖中的最短路徑。

1. **核心**：使用佇列 (Queue) 進行層次遍歷。
2. **流程**：
   - 使用 `visited` 集合記錄已訪問節點。
   - 將起點加入 `deque` 與 `visited`。
   - 每次從左側彈出一個節點，並將其所有未訪問的鄰居加入右側。

時間複雜度：O(V + E)，空間複雜度：O(V)。

```python
import collections

def bfs(graph, start):
    # graph: { node: [neighbors] }
    visited = set([start])
    queue = collections.deque([start])
    result = []

    while queue:
        node = queue.popleft()
        result.append(node)

        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
    
    return result

# 尋找最短路徑 (包含路徑記錄)
def bfs_shortest_path(graph, start, target):
    queue = collections.deque([(start, [start])])
    visited = set([start])

    while queue:
        (node, path) = queue.popleft()
        if node == target:
            return path

        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append((neighbor, path + [neighbor]))
    return None
```
