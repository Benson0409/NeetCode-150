# Dijkstra's Algorithm

## 解題思路

Dijkstra 演算法用於在加權圖中尋找單源最短路徑。

1. **核心**：使用最小堆 (Min-Heap) 始終選擇當前距離起點最近的未訪問節點。
2. **流程**：
   - 初始化 `distances` 字典，起點為 0，其餘為 `infinity`。
   - 將 `(0, start_node)` 放入堆中。
   - 當堆不為空時，彈出 `(dist, current_node)`：
     - 若 `dist` 大於已記錄的最短距離，跳過。
     - 遍歷鄰居，計算 `new_dist = dist + edge_weight`。
     - 若 `new_dist` 小於鄰居已記錄的距離，更新並加入堆中。

時間複雜度：O(E log V)，空間複雜度：O(V + E)。

```python
import heapq

def dijkstra(graph, start):
    # graph: { node: [(neighbor, weight), ...] }
    distances = {node: float('inf') for node in graph}
    distances[start] = 0
    pq = [(0, start)]  # (distance, node)

    while pq:
        current_dist, current_node = heapq.heappop(pq)

        if current_dist > distances[current_node]:
            continue

        for neighbor, weight in graph[current_node]:
            distance = current_dist + weight

            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(pq, (distance, neighbor))

    return distances
```
