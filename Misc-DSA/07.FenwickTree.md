# Fenwick Tree (Binary Indexed Tree)

Fenwick Tree（BIT）是一種高效的前綴和資料結構，支援單點更新與前綴查詢，比線段樹程式碼更簡潔。

核心操作：`LSOne(i) = i & (-i)`，取得最低位的 1。

## 解題思路

- `update(i, delta)`：更新位置 i，沿 i += LSOne(i) 向上傳播
- `query(i)`：查詢 `[1, i]` 的前綴和，沿 i -= LSOne(i) 累加
- 區間和 `[l, r]` = `query(r) - query(l-1)`

時間複雜度：update/query O(log N)；空間複雜度：O(N)。

```cpp
class FenwickTree {
    vector<int> ft;
    vector<int> nums;
    int n;

    int lsOne(int i) { return i & (-i); }

public:
    FenwickTree(vector<int>& arr) : n(arr.size()), ft(arr.size() + 1, 0), nums(arr) {
        for (int j = 0; j < n; j++)
            for (int i = j + 1; i <= n; i += lsOne(i))
                ft[i] += arr[j];
    }

    int prefixSum(int i) {  // 1-indexed
        int sum = 0;
        for (; i > 0; i -= lsOne(i)) sum += ft[i];
        return sum;
    }

    int rangeSum(int l, int r) {  // 0-indexed [l, r]
        return prefixSum(r + 1) - prefixSum(l);
    }

    void update(int idx, int val) {  // 0-indexed
        int delta = val - nums[idx];
        nums[idx] = val;
        for (int i = idx + 1; i <= n; i += lsOne(i))
            ft[i] += delta;
    }
};
```

## Complexity Analysis
- Time Complexity: update/query O(log N)
- Space Complexity: O(N)
