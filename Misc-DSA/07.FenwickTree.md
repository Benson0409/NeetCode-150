# Fenwick Tree (Binary Indexed Tree)

## 解題思路

Fenwick Tree (BIT) 是一種比線段樹更輕量級的資料結構，用於處理動態前綴和與單點更新。

1. **核心**：利用整數的二進位表示法。每個節點儲存一個區間的加總。
2. **關鍵操作**：`LSOne(i) = i & (-i)` 取得最低位的 1（Lowbit）。
3. **流程**：
   - `update(i, delta)`：從 `i` 開始，每次加上 `LSOne(i)`，向上更新父節點。
   - `query(i)`：從 `i` 開始，每次減去 `LSOne(i)`，向下累加結果直到 0。

時間複雜度：O(log N)，空間複雜度：O(N)。

```python
class FenwickTree:
    def __init__(self, n):
        self.tree = [0] * (n + 1)

    def add(self, i, delta):
        # 1-indexed
        while i < len(self.tree):
            self.tree[i] += delta
            i += i & (-i)

    def query(self, i):
        # 1-indexed, returns sum[1...i]
        sum_ = 0
        while i > 0:
            sum_ += self.tree[i]
            i -= i & (-i)
        return sum_

    def range_query(self, l, r):
        # 1-indexed, returns sum[l...r]
        return self.query(r) - self.query(l - 1)
```
