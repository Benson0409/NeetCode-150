# 130. Surrounded Regions
Medium

Given an m x n matrix board containing 'X' and 'O', capture all regions that are 4-directionally surrounded by 'X'.

A region is captured by flipping all 'O's into 'X's in that surrounded region.

 

Example 1:
```
Input: board = [
["X","X","X","X"],
["X","O","O","X"],
["X","X","O","X"],
["X","O","X","X"]]
Output: [
["X","X","X","X"],
["X","X","X","X"],
["X","X","X","X"],
["X","O","X","X"]]
```

Constraints:
- m == board.length
- n == board[i].length

## 解題思路

從邊界出發標記「不可被包圍的 O」：

1. 凡是與邊界相連的 `'O'`（以及與其相連的 `'O'`）都不會被翻轉。
2. 遍歷矩陣的四條邊，若遇到 `'O'` 則進行 DFS，將所有連通的 `'O'` 暫時標記為另一個字元（例如 `'T'`）。
3. 標記完成後，再次遍歷整個矩陣：
   - 剩下的 `'O'` 都是被 `'X'` 包圍的 → 翻轉為 `'X'`。
   - 標記為 `'T'` 的格子都是安全的 → 還原為 `'O'`。

時間複雜度：O(m × n)，空間複雜度：O(m × n)。

```python
class Solution:
    def solve(self, board: List[List[str]]) -> None:
        ROWS, COLS = len(board), len(board[0])

        def capture(r, c):
            if r < 0 or c < 0 or r == ROWS or c == COLS or board[r][c] != "O":
                return
            board[r][c] = "T"
            capture(r + 1, c)
            capture(r - 1, c)
            capture(r, c + 1)
            capture(r, c - 1)

        # 1. (DFS) Capture unsurrounded regions (O -> T)
        for r in range(ROWS):
            for c in range(COLS):
                if board[r][c] == "O" and (r in [0, ROWS - 1] or c in [0, COLS - 1]):
                    capture(r, c)

        # 2. Capture surrounded regions (O -> X)
        for r in range(ROWS):
            for c in range(COLS):
                if board[r][c] == "O":
                    board[r][c] = "X"

        # 3. Uncapture unsurrounded regions (T -> O)
        for r in range(ROWS):
            for c in range(COLS):
                if board[r][c] == "T":
                    board[r][c] = "O"
```

## Complexity Analysis
- Time Complexity: O(m * n)
- Space Complexity: O(m * n)
