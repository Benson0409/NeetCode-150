# [130. Surrounded Regions](https://leetcode.com/problems/surrounded-regions/)
Medium

Given an m x n matrix board containing 'X' and 'O', capture all regions that are 4-directionally surrounded by 'X'.

A region is captured by flipping all 'O's into 'X's in that surrounded region.

 

Example 1:
```
Input: board = [
["X","X","X","X"],
["X","O","O","X"],
["X","X","O","X"],
["X","O","X","X"]]
Output: [
["X","X","X","X"],
["X","X","X","X"],
["X","X","X","X"],
["X","O","X","X"]]
```
Example 2:
```
Input: board = [["X"]]
Output: [["X"]]
 ```

Constraints:

- m == board.length
- n == board[i].length
- 1 <= m, n <= 200
- board[i][j] is 'X' or 'O'.

## 解題思路

從邊界出發標記「安全的 O」：

1. 凡是與邊界相連的 `'O'` 不會被翻轉，從四個邊界出發 DFS，標記這些格子
2. 掃描整個棋盤：未被標記且為 `'O'` 的格子翻轉為 `'X'`

時間複雜度：O(m × n)，空間複雜度：O(m × n)。

```cpp
class Solution {
    void dfs(vector<vector<char>>& board, int i, int j, vector<vector<bool>>& safe) {
        int m = board.size(), n = board[0].size();
        if (i < 0 || i >= m || j < 0 || j >= n || safe[i][j] || board[i][j] == 'X')
            return;
        safe[i][j] = true;
        dfs(board, i + 1, j, safe);
        dfs(board, i - 1, j, safe);
        dfs(board, i, j + 1, safe);
        dfs(board, i, j - 1, safe);
    }

public:
    void solve(vector<vector<char>>& board) {
        int m = board.size(), n = board[0].size();
        vector<vector<bool>> safe(m, vector<bool>(n, false));

        for (int j = 0; j < n; j++) {
            if (board[0][j] == 'O') dfs(board, 0, j, safe);
            if (board[m-1][j] == 'O') dfs(board, m - 1, j, safe);
        }
        for (int i = 0; i < m; i++) {
            if (board[i][0] == 'O') dfs(board, i, 0, safe);
            if (board[i][n-1] == 'O') dfs(board, i, n - 1, safe);
        }

        for (int i = 0; i < m; i++)
            for (int j = 0; j < n; j++)
                if (!safe[i][j] && board[i][j] == 'O')
                    board[i][j] = 'X';
    }
};
```

## Complexity Analysis
- Time Complexity: O(m × n)
- Space Complexity: O(m × n)
