# 191. Number of 1 Bits
Easy

Write a function that takes an unsigned integer and returns the number of '1' bits it has (also known as the Hamming weight).

 

Example 1:
```
Input: n = 11 (binary 1011)
Output: 3
```

Constraints:
- The input must be a 32-bit integer.

## 解題思路

方法一：逐位檢查
- 透過 `n % 2` 取得最後一位是否為 1，然後 `n >> 1` 右移。

方法二：Brian Kernighan's 演算法 (更優)
- `n & (n - 1)` 每次運算都會消除二進制中最右邊的「1」。
- 例如 `n = 1011`, `n - 1 = 1010` -> `n & (n - 1) = 1010`。
- 重複此動作直到 `n` 變為 0，運算的次數即為 1 的個數。

時間複雜度：O(1)（對 32 位整數來說最快為 1 的個數次），空間複雜度：O(1)。

```python
class Solution:
    def hammingWeight(self, n: int) -> int:
        res = 0
        while n:
            n &= n - 1
            res += 1
        return res
```

## Complexity Analysis
- Time Complexity: O(1) - specifically O(k) where k is the number of 1-bits.
- Space Complexity: O(1)
