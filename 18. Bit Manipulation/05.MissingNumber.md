# 268. Missing Number
Easy

Given an array nums containing n distinct numbers in the range [0, n], return the only number in the range that is missing from the array.

 

Example 1:
```
Input: nums = [3,0,1]
Output: 2
```

Constraints:
- n == nums.length
- 0 <= nums[i] <= n

## 解題思路

方法一：XOR 位元運算
- 原理與 Single Number 相同。將 `[0, n]` 的所有數字與 `nums` 中的所有數字全部進行 XOR。
- 出現兩次的數字會抵消，最後剩下的就是缺失的那個數字。

方法二：數學加總法 (更直觀)
- 計算 `0` 到 `n` 的理想總和：`sum = n * (n + 1) / 2`。
- 減去 `nums` 陣列的實際總和。
- 差值即為缺失的數字。

時間複雜度：O(N)，空間複雜度：O(1)。

```python
class Solution:
    def missingNumber(self, nums: List[int]) -> int:
        res = len(nums)

        for i in range(len(nums)):
            res += i - nums[i]

        return res
```

## Complexity Analysis
- Time Complexity: O(n)
- Space Complexity: O(1)
