# 136. Single Number
Easy

Given a non-empty array of integers `nums`, every element appears twice except for one. Find that single one. You must implement a solution with a linear runtime complexity and use only constant extra space.

 

Example 1:
```
Input: nums = [2,2,1]
Output: 1
```

Constraints:
- 1 <= nums.length <= 3 * 10^4
- Each element appears twice except for one.

## 解題思路

XOR 位元運算的特性：

1. `a ^ a = 0`（任何數與自己 XOR 結果為 0）。
2. `a ^ 0 = a`（任何數與 0 XOR 結果仍為該數）。
3. `a ^ b ^ a = (a ^ a) ^ b = 0 ^ b = b`（符合交換律與結合律）。

因此，只要將陣列中所有數字依序進行 XOR 運算，成對出現的數字都會互相抵消變成 0，最後剩下的結果就是那個只出現一次的數字。

時間複雜度：O(N)，空間複雜度：O(1)。

```python
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        res = 0
        for n in nums:
            res = n ^ res
        return res
```

## Complexity Analysis
- Time Complexity: O(n)
- Space Complexity: O(1)
