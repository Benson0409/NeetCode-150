# 371. Sum of Two Integers
Medium

Given two integers a and b, return the sum of the two integers without using the operators + and -.

 

Example 1:
```
Input: a = 1, b = 2
Output: 3
```

Constraints:
- -1000 <= a, b <= 1000

## 解題思路

位元加法模擬：

1. **XOR (a ^ b)**：執行「不進位」的加法。
2. **AND & 移位 (a & b << 1)**：取得「進位」的部分。
3. 重複上述過程，直到進位為 0。

**Python 特殊處理**：
- Python 的整數是無限精度的，不會像 C++ 的 32-bit 整數那樣自動溢位。
- 處理負數時，我們需要使用遮罩 `mask = 0xFFFFFFFF` 來模擬 32 位整數的行為。
- 最後若結果大於 `0x7FFFFFFF`，代表它是負數，需將其轉換回 Python 的負數表示法。

時間複雜度：O(1)（最多 32 次迭代），空間複雜度：O(1)。

```python
class Solution:
    def getSum(self, a: int, b: int) -> int:
        mask = 0xFFFFFFFF

        while (b & mask) > 0:
            carry = (a & b) << 1
            a = a ^ b
            b = carry

        # handle positive and negative numbers
        return (a & mask) if b > 0 else a
```

## Complexity Analysis
- Time Complexity: O(1)
- Space Complexity: O(1)
