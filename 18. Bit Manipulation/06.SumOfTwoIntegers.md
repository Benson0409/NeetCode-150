# [371. Sum of Two Integers](https://leetcode.com/problems/sum-of-two-integers/)
Medium

Given two integers a and b, return the sum of the two integers without using the operators + and -.

 

Example 1:
```
Input: a = 1, b = 2
Output: 3
```
Example 2:
```
Input: a = 2, b = 3
Output: 5
 ```

Constraints:

- -1000 <= a, b <= 1000

## 解題思路

位元加法模擬：

- `a ^ b`：不含進位的加法（XOR）
- `(a & b) << 1`：進位部分（AND 後左移一位）
- 重複上述步驟直到沒有進位（b == 0）

注意：C++ 中對負數位元操作可能有未定義行為，使用 `unsigned` 或 `uint32_t` 處理。

時間複雜度：O(1)（最多 32 次），空間複雜度：O(1)。

```cpp
class Solution {
public:
    int getSum(int a, int b) {
        while (b != 0) {
            unsigned carry = (unsigned)(a & b) << 1;
            a = a ^ b;
            b = carry;
        }
        return a;
    }
};
```

## Complexity Analysis
- Time Complexity: O(1)
- Space Complexity: O(1)
