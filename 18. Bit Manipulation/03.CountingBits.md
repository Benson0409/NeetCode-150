# [338. Counting Bits](https://leetcode.com/problems/counting-bits/)
Easy

Given an integer n, return an array ans of length n + 1 such that for each i (0 <= i <= n), ans[i] is the number of 1's in the binary representation of i.

 

Example 1:
```
Input: n = 2
Output: [0,1,1]
```
Example 2:
```
Input: n = 5
Output: [0,1,1,2,1,2]
 ```

Constraints:

- 0 <= n <= 10^5

## 解題思路

DP：利用 `dp[i] = dp[i >> 1] + (i & 1)`（右移一位去掉最低位，加上最低位是否為 1）。

或者用 offset 法：下一個 2 的冪次更新 offset，`dp[i] = 1 + dp[i - offset]`。

時間複雜度：O(N)，空間複雜度：O(N)。

```cpp
class Solution {
public:
    vector<int> countBits(int n) {
        vector<int> dp(n + 1, 0);
        for (int i = 1; i <= n; i++)
            dp[i] = dp[i >> 1] + (i & 1);
        return dp;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N)
- Space Complexity: O(N)
