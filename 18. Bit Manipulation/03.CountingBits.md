# 338. Counting Bits
Easy

Given an integer n, return an array ans of length n + 1 such that for each i (0 <= i <= n), ans[i] is the number of 1's in the binary representation of i.

 

Example 1:
```
Input: n = 2
Output: [0,1,1]
```

Constraints:
- 0 <= n <= 10^5

## 解題思路

動態規劃 (DP)：

1. 我們可以觀察到：一個數字 `i` 的 1 的個數，與它的二倍、或是與它減去最高位後的數字有關係。
2. **方法：Offset**
   - 記錄目前的最高位權重 `offset` (如 1, 2, 4, 8, ...)。
   - 當 `i == offset * 2` 時，更新 `offset = i`。
   - `dp[i] = 1 + dp[i - offset]`。
   - 例如：`dp[6] = 1 + dp[6 - 4] = 1 + dp[2] = 1 + 1 = 2`。

時間複雜度：O(N)，空間複雜度：O(1)（除結果陣列外）。

```python
class Solution:
    def countBits(self, n: int) -> List[int]:
        dp = [0] * (n + 1)
        offset = 1

        for i in range(1, n + 1):
            if offset * 2 == i:
                offset = i
            dp[i] = 1 + dp[i - offset]

        return dp
```

## Complexity Analysis
- Time Complexity: O(n)
- Space Complexity: O(1) - (not counting result)
