# 190. Reverse Bits
Easy

Reverse bits of a given 32 bits unsigned integer.

 

Example 1:
```
Input: n = 00000010100101000001111010011100
Output:    964176192 (00111001011110000010100101000000)
```

Constraints:
- The input must be a 32-bit integer.

## 解題思路

逐位翻轉：

1. 建立一個結果變數 `res = 0`。
2. 進行 32 次迴圈（因為是 32 位整數）：
   - 使用 `(n >> i) & 1` 取得 `n` 的第 `i` 位數字。
   - 將取得的位元向左移位，放到鏡像位置：`bit << (31 - i)`。
   - 將結果使用 OR (`|`) 累加到 `res` 中。

時間複雜度：O(1)（固定 32 次），空間複雜度：O(1)。

```python
class Solution:
    def reverseBits(self, n: int) -> int:
        res = 0
        for i in range(32):
            bit = (n >> i) & 1
            res = res | (bit << (31 - i))
        return res
```

## Complexity Analysis
- Time Complexity: O(1)
- Space Complexity: O(1)
