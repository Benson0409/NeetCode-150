# [621. Task Scheduler](https://leetcode.com/problems/task-scheduler/)
Medium

Given a characters array tasks, representing the tasks a CPU needs to do, where each letter represents a different task. Tasks could be done in any order. Each task is done in one unit of time. For each unit of time, the CPU could complete either one task or just be idle.

However, there is a non-negative integer n that represents the cooldown period between two same tasks (the same letter in the array), that is that there must be at least n units of time between any two same tasks.

Return the least number of units of times that the CPU will take to finish all the given tasks.

 

Example 1:
```
Input: tasks = ["A","A","A","B","B","B"], n = 2
Output: 8
Explanation: 
A -> B -> idle -> A -> B -> idle -> A -> B
There is at least 2 units of time between any two same tasks.
```
Example 2:
```
Input: tasks = ["A","A","A","B","B","B"], n = 0
Output: 6
```
Example 3:
```
Input: tasks = ["A","A","A","A","A","A","B","C","D","E","F","G"], n = 2
Output: 16
 ```

Constraints:

- 1 <= task.length <= 104
- tasks[i] is upper-case English letter.
- The integer n is in the range [0, 100].

## 解題思路

數學公式法（不需真正模擬）：

- 找出出現次數最多的任務頻率 `max`，以及有多少種任務有這個最大頻率 `maxCount`
- 以最頻繁的任務為框架，排列成 `(max-1)` 個「長度為 n+1 的 slot」，最後再加 `maxCount` 個任務
- 空閒時間 = `max(0, 空 slot 數 - 可填入的其他任務數)`
- 答案 = `tasks.length + idleTime`

若其他任務足夠多，idle 為 0，答案就是 `tasks.length`。

時間複雜度：O(N)，空間複雜度：O(26)。

```cpp
class Solution {
public:
    int leastInterval(vector<char>& tasks, int n) {
        int freq[26] = {};
        int maxFreq = 0, maxCount = 0;

        for (char t : tasks) {
            freq[t - 'A']++;
            if (freq[t - 'A'] == maxFreq) {
                maxCount++;
            } else if (freq[t - 'A'] > maxFreq) {
                maxFreq = freq[t - 'A'];
                maxCount = 1;
            }
        }

        int blankSeq = maxFreq - 1;           // 有幾段冷卻空間
        int blankSeqLen = n - (maxCount - 1); // 每段空間可以放幾個 idle
        int emptySlots = blankSeq * blankSeqLen;
        int availableSlots = (int)tasks.size() - (maxCount * maxFreq);
        int idleTime = max(0, emptySlots - availableSlots);

        return (int)tasks.size() + idleTime;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N)
- Space Complexity: O(26) = O(1)
