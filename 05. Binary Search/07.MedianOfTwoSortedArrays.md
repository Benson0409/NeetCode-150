# [4. Median of Two Sorted Arrays](https://leetcode.com/problems/median-of-two-sorted-arrays/)
Hard

Given two sorted arrays nums1 and nums2 of size m and n respectively, return the median of the two sorted arrays.

The overall run time complexity should be O(log (m+n)).

 

Example 1:
```
Input: nums1 = [1,3], nums2 = [2]
Output: 2.00000
Explanation: merged array = [1,2,3] and median is 2.
```
Example 2:
```
Input: nums1 = [1,2], nums2 = [3,4]
Output: 2.50000
Explanation: merged array = [1,2,3,4] and median is (2 + 3) / 2 = 2.5.
 ```

Constraints:
- nums1.length == m
- nums2.length == n
- 0 <= m <= 1000
- 0 <= n <= 1000
- 1 <= m + n <= 2000
- -10<sup>6</sup> <= nums1[i], nums2[i] <= 10<sup>6</sup>

## 解題思路

對較短的陣列做二分搜尋，找到合適的「切割點」使左半部所有元素 ≤ 右半部所有元素：

- 確保 `nums1` 是較短的陣列（交換若必要）
- `half = (total + 1) / 2`：左半部總共需要的元素數
- 二分搜尋 `i`（nums1 取幾個元素），則 `j = half - i`（nums2 取幾個）
- 有效切割條件：`nums1L <= nums2R && nums2L <= nums1R`
  - 若偶數總長：中位數 = `(max(nums1L, nums2L) + min(nums1R, nums2R)) / 2.0`
  - 若奇數總長：中位數 = `max(nums1L, nums2L)`

時間複雜度：O(log(min(m, n)))，空間複雜度：O(1)。

```cpp
class Solution {
public:
    double findMedianSortedArrays(vector<int>& nums1, vector<int>& nums2) {
        if (nums1.size() > nums2.size())
            return findMedianSortedArrays(nums2, nums1);

        int total = nums1.size() + nums2.size();
        int half = (total + 1) / 2;
        int l = 0, r = (int)nums1.size();

        while (l <= r) {
            int i = l + (r - l) / 2;
            int j = half - i;

            int nums1L = (i > 0) ? nums1[i-1] : INT_MIN;
            int nums1R = (i < (int)nums1.size()) ? nums1[i] : INT_MAX;
            int nums2L = (j > 0) ? nums2[j-1] : INT_MIN;
            int nums2R = (j < (int)nums2.size()) ? nums2[j] : INT_MAX;

            if (nums1L <= nums2R && nums2L <= nums1R) {
                if (total % 2 == 0)
                    return (max(nums1L, nums2L) + min(nums1R, nums2R)) / 2.0;
                else
                    return max(nums1L, nums2L);
            } else if (nums1L > nums2R) {
                r = i - 1;
            } else {
                l = i + 1;
            }
        }
        return 0.0;
    }
};
```

## Complexity Analysis
- Time Complexity: O(log(min(m, n)))
- Space Complexity: O(1)
