# [309. Best Time to Buy and Sell Stock with Cooldown](https://leetcode.com/problems/best-time-to-buy-and-sell-stock-with-cooldown/)
Medium

You are given an array prices where prices[i] is the price of a given stock on the ith day.

Find the maximum profit you can achieve. You may complete as many transactions as you like with the following restrictions:

After you sell your stock, you cannot buy stock on the next day (i.e., cooldown one day).

 

Example 1:
```
Input: prices = [1,2,3,0,2]
Output: 3
```
Example 2:
```
Input: prices = [1]
Output: 0
``` 

Constraints:

- 1 <= prices.length <= 5000
- 0 <= prices[i] <= 1000

## 解題思路

狀態機 DP（三個狀態）：

- `hold`：持有股票時的最大利潤
- `sold`：剛賣出（下一天進入 cooldown）
- `rest`：cooldown 或空閒狀態

狀態轉移：
- `hold = max(hold, rest - prices[i])`（繼續持有 or 從 rest 狀態買入）
- `rest = max(rest, sold)`（繼續 rest or 從 sold 轉入）
- `sold = hold_prev + prices[i]`（賣出，用上一輪的 hold）

時間複雜度：O(N)，空間複雜度：O(1)。

```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int hold = INT_MIN, sold = 0, rest = 0;
        for (int p : prices) {
            int prevHold = hold;
            hold = max(hold, rest - p);
            rest = max(rest, sold);
            sold = prevHold + p;
        }
        return max(sold, rest);
    }
};
```

## Complexity Analysis
- Time Complexity: O(N)
- Space Complexity: O(1)
