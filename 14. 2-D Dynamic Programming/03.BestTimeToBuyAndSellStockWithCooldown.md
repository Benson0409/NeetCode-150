# 309. Best Time to Buy and Sell Stock with Cooldown
Medium

You are given an array prices where prices[i] is the price of a given stock on the ith day. Find the maximum profit you can achieve. After you sell your stock, you cannot buy stock on the next day (i.e., cooldown one day).

 

Example 1:
```
Input: prices = [1,2,3,0,2]
Output: 3
```

Constraints:
- 1 <= prices.length <= 5000

## 解題思路

DFS + 記憶化（Memoization）：

1. 每一天有兩個狀態：**買入 (Buying)** 或 **賣出 (Selling)**。
2. 決策過程：
   - 若目前狀態是 **Buying**：
     - 可以選擇「買」：獲利減去當前股價，進入下一天，狀態變為 Selling。
     - 可以選擇「不買」：獲利不變，進入下一天，狀態仍為 Buying。
   - 若目前狀態是 **Selling**：
     - 可以選擇「賣」：獲利加上當前股價，進入「後天」（因為有一天 cooldown），狀態變回 Buying。
     - 可以選擇「不賣」：獲利不變，進入下一天，狀態仍為 Selling。
3. 使用一個字典 `dp` 記錄 `(index, buying)` 的結果以避免重複計算。

時間複雜度：O(N)，空間複雜度：O(N)。

```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        # State: Buying or Selling
        # If Buy -> i + 1
        # If Sell -> i + 2

        dp = {}  # (i, buying) : max_profit

        def dfs(i, buying):
            if i >= len(prices):
                return 0
            if (i, buying) in dp:
                return dp[(i, buying)]

            cooldown = dfs(i + 1, buying)
            if buying:
                buy = dfs(i + 1, not buying) - prices[i]
                dp[(i, buying)] = max(buy, cooldown)
            else:
                sell = dfs(i + 2, not buying) + prices[i]
                dp[(i, buying)] = max(sell, cooldown)

            return dp[(i, buying)]

        return dfs(0, True)
```

## Complexity Analysis
- Time Complexity: O(n)
- Space Complexity: O(n)
