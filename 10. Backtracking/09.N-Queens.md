# [51. N-Queens](https://leetcode.com/problems/n-queens/)
Hard

The n-queens puzzle is the problem of placing n queens on an n x n chessboard such that no two queens attack each other.

Given an integer n, return all distinct solutions to the n-queens puzzle. You may return the answer in any order.

Each solution contains a distinct board configuration of the n-queens' placement, where 'Q' and '.' both indicate a queen and an empty space, respectively.

 

Example 1:
```
Input: n = 4
Output: [[".Q..","...Q","Q...","..Q."],["..Q.","Q...","...Q",".Q.."]]
Explanation: There exist two distinct solutions to the 4-queens puzzle as shown above
```
Example 2:
```
Input: n = 1
Output: [["Q"]]
``` 

Constraints:
- 1 <= n <= 9

## 解題思路

逐列回溯法：

- 對每一行（column），嘗試在每一列（row）放置皇后
- 用三個布林陣列記錄攻擊範圍：
  - `rw[r]`：第 r 列已被佔用
  - `ld[r-c+n-1]`：左對角線已被佔用（`r-c` 可為負，加 `n-1` 偏移）
  - `rd[r+c]`：右對角線已被佔用
- 若三個條件都未被佔用，則放置皇后，遞迴處理下一行，回溯時取消標記
- 所有行都放置完畢時，將棋盤轉換為字串列表加入結果

時間複雜度：O(N!)，空間複雜度：O(N²)（建立結果字串）。

```cpp
class Solution {
    vector<int> row;
    vector<bool> rw, ld, rd;

    vector<string> buildBoard(int n) {
        vector<string> res(n, string(n, '.'));
        for (int i = 0; i < n; i++)
            res[i][row[i]] = 'Q';
        return res;
    }

    void backtrack(int c, int n, vector<vector<string>>& ans) {
        if (c == n) {
            ans.push_back(buildBoard(n));
            return;
        }
        for (int r = 0; r < n; r++) {
            if (!rw[r] && !ld[r - c + n - 1] && !rd[r + c]) {
                rw[r] = ld[r - c + n - 1] = rd[r + c] = true;
                row[c] = r;
                backtrack(c + 1, n, ans);
                rw[r] = ld[r - c + n - 1] = rd[r + c] = false;
            }
        }
    }

public:
    vector<vector<string>> solveNQueens(int n) {
        row.resize(n);
        rw.assign(n, false);
        ld.assign(2 * n - 1, false);
        rd.assign(2 * n - 1, false);
        vector<vector<string>> ans;
        backtrack(0, n, ans);
        return ans;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N!)
- Space Complexity: O(N²)
