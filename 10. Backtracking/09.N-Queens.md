# 51. N-Queens
Hard

The n-queens puzzle is the problem of placing n queens on an n x n chessboard such that no two queens attack each other.

Given an integer n, return all distinct solutions to the n-queens puzzle. You may return the answer in any order.

Each solution contains a distinct board configuration of the n-queens' placement, where 'Q' and '.' both indicate a queen and an empty space, respectively.

 

Example 1:
```
Input: n = 4
Output: [[".Q..","...Q","Q...","..Q."],["..Q.","Q...","...Q",".Q.."]]
```

Constraints:
- 1 <= n <= 9

## 解題思路

回溯法，逐列放置皇后並記錄受攻擊範圍：

1. 維持三個集合來記錄目前被佔用的行與對角線：
   - `col`: 被佔用的縱向行。
   - `posDiag`: 正對角線 `(r + c)`。
   - `negDiag`: 負對角線 `(r - c)`。
2. 逐列進行 DFS：對於每一行 `r`，若其所在列、正對角線、負對角線皆未被佔用，則放置皇后。
3. 遞迴至下一列。
4. 回溯：移除皇后並將對應位置從三個集合中刪除。
5. 當所有列都放置完畢時，根據當前棋盤狀態建立字串列表並存入結果。

時間複雜度：O(n!)，空間複雜度：O(n²)。

```python
class Solution:
    def solveNQueens(self, n: int) -> List[List[str]]:
        col = set()
        posDiag = set()  # (r + c)
        negDiag = set()  # (r - c)

        res = []
        board = [["."] * n for i in range(n)]

        def backtrack(r):
            if r == n:
                copy = ["".join(row) for row in board]
                res.append(copy)
                return

            for c in range(n):
                if c in col or (r + c) in posDiag or (r - c) in negDiag:
                    continue

                col.add(c)
                posDiag.add(r + c)
                negDiag.add(r - c)
                board[r][c] = "Q"

                backtrack(r + 1)

                col.remove(c)
                posDiag.remove(r + c)
                negDiag.remove(r - c)
                board[r][c] = "."

        backtrack(0)
        return res
```

## Complexity Analysis
- Time Complexity: O(n!)
- Space Complexity: O(n^2)
