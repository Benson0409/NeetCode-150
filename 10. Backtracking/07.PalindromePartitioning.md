# 131. Palindrome Partitioning
Medium

Given a string s, partition s such that every substring of the partition is a palindrome. Return all possible palindrome partitioning of s.

 

Example 1:
```
Input: s = "aab"
Output: [["a","a","b"],["aa","b"]]
```

Constraints:
- 1 <= s.length <= 16

## 解題思路

回溯法：

1. 從字串開頭嘗試所有可能的切割點。
2. 若切割出的左側子字串是回文（Palindrome），則將其加入當前路徑 `part`。
3. 對剩餘的右側子字串進行遞迴。
4. 若處理完所有字元，將 `part` 的副本加入結果中。
5. 回溯：移除最後加入的子字串，嘗試下一個切割點。

時間複雜度：O(2^n · n)，空間複雜度：O(n)。

```python
class Solution:
    def partition(self, s: str) -> List[List[str]]:
        res = []
        part = []

        def dfs(i):
            if i >= len(s):
                res.append(part.copy())
                return

            for j in range(i, len(s)):
                if self.isPali(s, i, j):
                    part.append(s[i : j + 1])
                    dfs(j + 1)
                    part.pop()

        dfs(0)
        return res

    def isPali(self, s, l, r):
        while l < r:
            if s[l] != s[r]:
                return False
            l, r = l + 1, r - 1
        return True
```

## Complexity Analysis
- Time Complexity: O(2^n)
- Space Complexity: O(n)
