# [131. Palindrome Partitioning](https://leetcode.com/problems/palindrome-partitioning/)
Medium

Given a string s, partition s such that every substring of the partition is a palindrome. Return all possible palindrome partitioning of s.

A palindrome string is a string that reads the same backward as forward.

 

Example 1:
```
Input: s = "aab"
Output: [["a","a","b"],["aa","b"]]
```
Example 2:
```
Input: s = "a"
Output: [["a"]]
 ```

Constraints:

- 1 <= s.length <= 16
- s contains only lowercase English letters.

## 解題思路

回溯法：

1. 從字串開頭嘗試所有可能的切割點 `i`（長度 1 到剩餘字串長度）
2. 若 `s[0..i-1]` 是回文 → 加入當前組合，遞迴處理剩餘字串 `s[i..]`
3. 回溯時移除最後加入的子字串
4. 若字串為空 → 加入結果

時間複雜度：O(N × 2^N)，空間複雜度：O(N)。

```cpp
class Solution {
    bool isPalindrome(const string& s) {
        int l = 0, r = (int)s.size() - 1;
        while (l < r)
            if (s[l++] != s[r--]) return false;
        return true;
    }

    void helper(const string& s, vector<string>& step, vector<vector<string>>& result) {
        if (s.empty()) {
            result.push_back(step);
            return;
        }
        for (int i = 1; i <= (int)s.size(); i++) {
            string temp = s.substr(0, i);
            if (isPalindrome(temp)) {
                step.push_back(temp);
                helper(s.substr(i), step, result);
                step.pop_back();
            }
        }
    }

public:
    vector<vector<string>> partition(string s) {
        vector<vector<string>> result;
        vector<string> step;
        helper(s, step, result);
        return result;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N × 2^N)
- Space Complexity: O(N)
