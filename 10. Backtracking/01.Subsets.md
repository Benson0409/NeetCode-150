# [78. Subsets](https://leetcode.com/problems/subsets/)
Medium

Given an integer array nums of unique elements, return all possible subsets (the power set).

The solution set must not contain duplicate subsets. Return the solution in any order.

 
Example 1:
```
Input: nums = [1,2,3]
Output: [[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]
```
Example 2:
```
Input: nums = [0]
Output: [[],[0]]
 ```

Constraints:

- 1 <= nums.length <= 10
- -10 <= nums[i] <= 10
- All the numbers of nums are unique.

## 解題思路

回溯法（Backtracking）：對每個元素做「選」或「不選」兩種決策：

1. 「選」：加入當前元素，遞迴處理下一個索引
2. 回溯後「不選」：移除元素，遞迴處理下一個索引
3. 到達陣列末端時，將當前組合加入結果

共有 2^N 種子集，時間複雜度 O(2^N)，空間複雜度 O(N)（遞迴深度）。

```cpp
class Solution {
    void backtrack(vector<int>& nums, int index, vector<int>& tmp, vector<vector<int>>& ans) {
        if (index == (int)nums.size()) {
            ans.push_back(tmp);
            return;
        }
        // 選
        tmp.push_back(nums[index]);
        backtrack(nums, index + 1, tmp, ans);
        // 不選
        tmp.pop_back();
        backtrack(nums, index + 1, tmp, ans);
    }

public:
    vector<vector<int>> subsets(vector<int>& nums) {
        vector<vector<int>> ans;
        vector<int> tmp;
        backtrack(nums, 0, tmp, ans);
        return ans;
    }
};
```

## Complexity Analysis
- Time Complexity: O(2^N)
- Space Complexity: O(N)
