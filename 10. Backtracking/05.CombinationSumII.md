# [40. Combination Sum II](https://leetcode.com/problems/combination-sum-ii/)
Medium

Given a collection of candidate numbers (candidates) and a target number (target), find all unique combinations in candidates where the candidate numbers sum to target.

Each number in candidates may only be used once in the combination.

Note: The solution set must not contain duplicate combinations.

 

Example 1:
```
Input: candidates = [10,1,2,7,6,1,5], target = 8
Output: 
[
[1,1,6],
[1,2,5],
[1,7],
[2,6]
]
```
Example 2:
```
Input: candidates = [2,5,2,1,2], target = 5
Output: 
[
[1,2,2],
[5]
]
 ```

Constraints:

- 1 <= candidates.length <= 100
- 1 <= candidates[i] <= 50
- 1 <= target <= 30

## 解題思路

與 Combination Sum I 類似，但每個元素只能用一次，且有重複元素需去除重複組合。

策略（與 Subsets II 相同去重技巧）：

1. 先排序
2. 回溯中：「選」`nums[index]` 後用 `index+1` 遞迴（不允許重複選）
3. 「不選」時：跳過所有連續相同的值後再遞迴

時間複雜度：O(2^N)，空間複雜度：O(N)。

```cpp
class Solution {
    void backtrack(vector<int>& nums, int index, int target, vector<int>& tmp, vector<vector<int>>& ans) {
        if (target == 0) {
            ans.push_back(tmp);
            return;
        }
        if (index >= (int)nums.size() || target < 0) return;

        // 選
        tmp.push_back(nums[index]);
        backtrack(nums, index + 1, target - nums[index], tmp, ans);
        tmp.pop_back();
        // 不選：跳過重複值
        while (index + 1 < (int)nums.size() && nums[index] == nums[index + 1])
            index++;
        backtrack(nums, index + 1, target, tmp, ans);
    }

public:
    vector<vector<int>> combinationSum2(vector<int>& candidates, int target) {
        sort(candidates.begin(), candidates.end());
        vector<vector<int>> ans;
        vector<int> tmp;
        backtrack(candidates, 0, target, tmp, ans);
        return ans;
    }
};
```

## Complexity Analysis
- Time Complexity: O(2^N)
- Space Complexity: O(N)
