# [39. Combination Sum](https://leetcode.com/problems/combination-sum/)
Medium

Given an array of distinct integers candidates and a target integer target, return a list of all unique combinations of candidates where the chosen numbers sum to target. You may return the combinations in any order.

The same number may be chosen from candidates an unlimited number of times. Two combinations are unique if the frequency of at least one of the chosen numbers is different.

It is guaranteed that the number of unique combinations that sum up to target is less than 150 combinations for the given input.

 

Example 1:
```
Input: candidates = [2,3,6,7], target = 7
Output: [[2,2,3],[7]]
```
Example 2:
```
Input: candidates = [2,3,5], target = 8
Output: [[2,2,2,2],[2,3,3],[3,5]]
```
Example 3:
```
Input: candidates = [2], target = 1
Output: []
 ```

Constraints:

- 1 <= candidates.length <= 30
- 1 <= candidates[i] <= 200
- All elements of candidates are distinct.
- 1 <= target <= 500

## 解題思路

回溯法，每個元素可以重複選取（與 Subsets 不同）：

- 從 `index` 開始遍歷，每次選一個元素後仍從同一 `index` 遞迴（允許重複）
- 若 `target == 0` → 找到一組答案
- 若 `target < 0` 或越界 → 剪枝回退

```cpp
class Solution {
    void backtrack(vector<int>& nums, int index, int target, vector<int>& tmp, vector<vector<int>>& ans) {
        if (target == 0) {
            ans.push_back(tmp);
            return;
        }
        if (index >= (int)nums.size() || target < 0) return;

        for (int i = index; i < (int)nums.size(); i++) {
            tmp.push_back(nums[i]);
            backtrack(nums, i, target - nums[i], tmp, ans); // i 不是 i+1，允許重複選
            tmp.pop_back();
        }
    }

public:
    vector<vector<int>> combinationSum(vector<int>& candidates, int target) {
        vector<vector<int>> ans;
        vector<int> tmp;
        backtrack(candidates, 0, target, tmp, ans);
        return ans;
    }
};
```

## Complexity Analysis
- Time Complexity: O(N^(T/M))，T 為 target，M 為最小候選數
- Space Complexity: O(T/M)（遞迴深度）
