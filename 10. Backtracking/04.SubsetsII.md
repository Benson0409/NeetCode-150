# [90. Subsets II](https://leetcode.com/problems/subsets-ii/)
Medium

Given an integer array nums that may contain duplicates, return all possible subsets (the power set).

The solution set must not contain duplicate subsets. Return the solution in any order.

 

Example 1:
```
Input: nums = [1,2,2]
Output: [[],[1],[1,2],[1,2,2],[2],[2,2]]
```
Example 2:
```
Input: nums = [0]
Output: [[],[0]]
 ```

Constraints:

- 1 <= nums.length <= 10
- -10 <= nums[i] <= 10

## 解題思路

與 Subsets I 相同，但需去除重複子集。關鍵：**先排序，再跳過重複元素**。

在「不選」的分支決策前，跳過所有與當前元素相同的連續值（確保不從同一值的第二個開始重複生成子集）：

```
選 nums[index] → 遞迴
不選（跳過所有 nums[index] 的重複值）→ 遞迴
```

時間複雜度：O(2^N)，空間複雜度：O(N)。

```cpp
class Solution {
    void backtrack(vector<int>& nums, int index, vector<int>& tmp, vector<vector<int>>& ans) {
        if (index == (int)nums.size()) {
            ans.push_back(tmp);
            return;
        }
        // 選
        tmp.push_back(nums[index]);
        backtrack(nums, index + 1, tmp, ans);
        tmp.pop_back();
        // 不選：跳過重複值
        while (index + 1 < (int)nums.size() && nums[index] == nums[index + 1])
            index++;
        backtrack(nums, index + 1, tmp, ans);
    }

public:
    vector<vector<int>> subsetsWithDup(vector<int>& nums) {
        sort(nums.begin(), nums.end());
        vector<vector<int>> ans;
        vector<int> tmp;
        backtrack(nums, 0, tmp, ans);
        return ans;
    }
};
```

## Complexity Analysis
- Time Complexity: O(2^N)
- Space Complexity: O(N)
