# 79. Word Search
Medium

Given an m x n grid of characters board and a string word, return true if word exists in the grid.

The word can be constructed from letters of sequentially adjacent cells, where adjacent cells are horizontally or vertically neighboring. The same letter cell may not be used more than once.

 

Example 1:
```
Input: board = [["A","B","C","E"],["S","F","C","S"],["A","D","E","E"]], word = "ABCCED"
Output: true
```

Constraints:
- m == board.length
- n = board[i].length
- 1 <= word.length <= 15

## 解題思路

回溯法 + DFS：

1. 遍歷所有格子，尋找與 `word[0]` 相同的起點。
2. 從起點出發，向上下左右四個方向進行 DFS。
3. 傳遞目前匹配到的字元索引 `r`。
4. **避免重複使用格子**：進入 DFS 時，暫時將 `board[r][c]` 替換為空字元（或標記已訪問），並在遞迴結束後還原。
5. 若匹配長度等於 `word` 長度，則回傳 `True`。

時間複雜度：O(m × n × 4^L)，L 為單字長度。空間複雜度：O(L)。

```python
class Solution:
    def exist(self, board: List[List[str]], word: str) -> bool:
        ROWS, COLS = len(board), len(board[0])
        path = set()

        def dfs(r, c, i):
            if i == len(word):
                return True
            if (
                r < 0
                or c < 0
                or r >= ROWS
                or c >= COLS
                or word[i] != board[r][c]
                or (r, c) in path
            ):
                return False

            path.add((r, c))
            res = (
                dfs(r + 1, c, i + 1)
                or dfs(r - 1, c, i + 1)
                or dfs(r, c + 1, i + 1)
                or dfs(r, c - 1, i + 1)
            )
            path.remove((r, c))
            return res

        for r in range(ROWS):
            for c in range(COLS):
                if dfs(r, c, 0):
                    return True
        return False
```

## Complexity Analysis
- Time Complexity: O(n * m * 4^n)
- Space Complexity: O(n)
